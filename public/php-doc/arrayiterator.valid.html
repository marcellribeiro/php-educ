ArrayIterator::valid

(PHP 5, PHP 7, PHP 8)

ArrayIterator::valid — Verifica se o array possui mais elementos

Descrição

ArrayIterator::valid(): bool

Verifica se o array possui mais algum elemento.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Nenhum valor é retornado.

Exemplos

Exemplo #1 Exemplo de ArrayIterator::valid()

<?php
$array = array('1' => 'um');

$arrayobject = new ArrayObject($array);
$iterator = $arrayobject->getIterator();

var_dump($iterator->valid()); //bool(true)

$iterator->next(); // avança para o próximo item

//bool(false) porque só existe um único elemento no array
var_dump($iterator->valid());
?>

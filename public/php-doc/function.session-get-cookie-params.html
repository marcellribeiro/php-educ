session_get_cookie_params

(PHP 4, PHP 5, PHP 7, PHP 8)

session_get_cookie_params — Obtém os parâmetros do cookie da sessão

Descrição

session_get_cookie_params(): array

Obtém os parâmetros do cookie da sessão.

Valor Retornado

Retorna um array com informação do cookie da sessão atual. O array contém os seguintes items:

  • "lifetime" - A duração do cookie em segundos.
  • "path" - O caminho onde a informação está armazenada.
  • "domain" - O domínio do cookie.
  • "secure" - O cookie deve ser enviado apenas em conexões seguras.
  • "httponly" - O cookie pode ser acessado somente atráves do protocolo HTTP.

Changelog

Versão Descrição
5.2.0 A entrada "httponly" foi adicionada no array retornado.

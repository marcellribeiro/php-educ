Configurações em Execução

O comportamento dessas funções é afetado pelas configurações do php.ini.

Iconv Opções de Configuração
Nome Padrão Modificável Changelog
iconv.input_encoding "" PHP_INI_ALL Obsoleto no PHP 5.6.0.
iconv.output_encoding "" PHP_INI_ALL Obsoleto no PHP 5.6.0.
iconv.internal_encoding "" PHP_INI_ALL Obsoleto no PHP 5.6.0.

Aqui está uma breve explicação das diretivas de configuração.

Aviso

Alguns sistemas (como o IBM AIX) usam "ISO8859-1" em vez de "ISO-8859-1", então esse valor deve ser usado em opções de configuração e parâmetros de função.

iconv.input_encoding string
Aviso

Este recurso tornou-se OBSOLETO a partir do PHP 5.6.0. O uso deste recurso é altamente desaconselhado.

Os usuários do PHP 5.6 e posteriores devem deixar em branco e definir input_encoding .

iconv.output_encoding string
Aviso

Este recurso tornou-se OBSOLETO a partir do PHP 5.6.0. O uso deste recurso é altamente desaconselhado.

Os usuários do PHP 5.6 e posteriores devem deixar em branco e definir output_encoding

iconv.internal_encoding string
Aviso

Este recurso tornou-se OBSOLETO a partir do PHP 5.6.0. O uso deste recurso é altamente desaconselhado.

Os usuários do PHP 5.6 e posteriores devem deixar em branco e definir default_charset em seu lugar.

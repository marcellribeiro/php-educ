readfile

(PHP 4, PHP 5, PHP 7, PHP 8)

readfile — Lê e exibe o conteúdo de um arquivo

Descrição

readfile(string $nomedoarquivo, bool $use_include_path = ?, resource $context = ?): int

Lê um arquivo e escreve o seu conteúdo para o buffer de saída (output buffer).

Parâmetros

filename

O nome do arquivo a ser lido.

use_include_path

Você pode usar o segundo parâmetro opcional e o configurar para true, se você quiser procurar o arquivo no include_path também.

context

A context stream resource.

Valor Retornado

Retorna o número de bytes lidos do arquivo. Se um erro ocorrer, false é retornado a menos que a função seja chamada com @readfile(), um erro é imprimido.

Notas

Dica

Uma URL pode ser usada como um nome de arquivo com esta função se os wrappers fopen estiverem habilitados. Consulte a função fopen() para mais detalhes sobre como especificar o nome do arquivo. Consulte Protocolos e Wrappers suportados para informações sobre as capacidades de cada wrapper, notas de uso e informações sobre quaisquer variáveis predefinidas que eles possam fornecer.

Um resource de contexto de stream.

imagecreatetruecolor

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

imagecreatetruecolor — Cria uma nova imagem true color

Descrição

imagecreatetruecolor(int $x_size, int $y_size): resource

imagecreatetruecolor() retorna um identificador de imagem representando uma imagem preta de tamanho x_size por y_size.

Exemplo #1 Criando um novo stream de imagem GD e enviando a imagem.

<?php
header ("Content-type: image/png");
$im = @imagecreatetruecolor(50, 100)
or die("Cannot Initialize new GD image stream");
$text_color = imagecolorallocate($im, 233, 14, 91);
imagestring($im, 1, 5, 5, "A Simple Text String", $text_color);
imagepng($im);
imagedestroy($im);
?>

Nota:

Esta função não funcionará com arquivos de formato GIF.

Veja também imagedestroy() e imagecreate().

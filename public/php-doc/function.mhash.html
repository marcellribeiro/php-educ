mhash

(PHP 4, PHP 5, PHP 7, PHP 8)

mhash — Computa a hash

Descrição

mhash(int $hash, string $data, string $key = ?): string

mhash() aplica a função hash especificada por hash para o parâmetro data.

Parâmetros

hash

O hash id. Uma das constantes MHASH_XXX.

data

A entrada do usuário, como uma string.

key

Se especificada, a função irá retornar o HMAC resultante. HMAC é o hashing com chave (keyed) para autenticações de mensagens, ou simplesmente um digest de mensagens que depende na chave específicada. Nem todos os algoritmos suportados em mhash podem ser usados em modo HMAC.

Valor Retornado

Retorna o hash resultante (também chamado digest) ou HMAC como uma string, ou false em erros.

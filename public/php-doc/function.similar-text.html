similar_text

(PHP 4, PHP 5, PHP 7, PHP 8)

similar_text — Calcula a similaridade entre duas strings

Descrição

similar_text(string $first, string $second, float &$percent = ?): int

Calcula a similaridade entre duas strings como descrito em Programming Classics: Implementing the World's Best Algorithms by Oliver (ISBN 0-131-00413-1). Note que esta implementação não usa uma stack como no pseudo-código de Oliver, mas chamadas recursivas as quais podem ou não tornar todo o processo mais rápido. Note também que a complexidade deste algoritmo é O(N**3) aonde N é o tamanho da string mais longa.

Parâmetros

primeira

A primeira string.

segunda

A segunda string.

percentual

Passando uma referência como terceiro argumento, a função similar_text() irá calcular o percentual de similaridade para você.

Valor Retornado

Retorna o número de caracteres iguais em ambas as strings.

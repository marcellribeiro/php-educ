mysqli_stmt::close

mysqli_stmt_close

(PHP 5, PHP 7, PHP 8)

mysqli_stmt::close -- mysqli_stmt_close — Closes a prepared statement

Descrição

Estilo orientado a objetos

public mysqli_stmt::close(): true

Estilo procedural

mysqli_stmt_close(mysqli_stmt $statement): true

Closes a prepared statement. mysqli_stmt_close() also deallocates the statement handle. If the current statement has pending or unread results, this function cancels them so that the next query can be executed.

Parâmetros

statement

Somente no estilo procedural: Um objeto mysqli_stmt retornado por mysqli_stmt_init().

Valor Retornado

Sempre retorna true.

Changelog

Versão Descrição
8.0.0 This function now always returns true. Previously it returned false on failure.

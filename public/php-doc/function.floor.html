floor

(PHP 4, PHP 5, PHP 7, PHP 8)

floor — Arredonda frações para baixo

Descrição

floor(float $value): float

Retorna o próximo menor valor inteiro, arredondando o valor se necessário.

Parâmetros

valor

O número a ser arredondado

Valor Retornado

O valor arredondado para o próximo menor valor inteiro. O valor retornado de floor() é do tipo float porque o tipo float normalmente abrange mais números possíveis do que integer.

Exemplos

Exemplo #1 Exemplos de floor()

<?php
echo floor(4.3); // 4
echo floor(9.999); // 9
echo floor(-3.14); // -4
?>

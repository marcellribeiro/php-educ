Error::getPrevious

(PHP 7, PHP 8)

Error::getPrevious — Retorna o último Throwable

Descrição

final public Error::getPrevious(): ?Throwable

Retorna Throwable anterior (o terceiro parâmetro do Error::__construct()).

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna Throwable anterior, se disponível, ou null.

Exemplos

Exemplo #1 Exemplo do método Error::getPrevious()

Iterando e imprimindo a pilha de erro.

<?php
class MyCustomError extends Error {}

function doStuff() {
try {
throw new InvalidArgumentError("Um erro!", 112);
} catch(Error $e) {
throw new MyCustomError("Algo aconteceu.", 911, $e);
}
}


try {
doStuff();
} catch(Error $e) {
do {
printf("%s:%d %s (%d) [%s]\n", $e->getFile(), $e->getLine(), $e->getMessage(), $e->getCode(), get_class($e));
} while($e = $e->getPrevious());
}
?>

O exemplo acima produzirá algo semelhante a:

/home/bjori/ex.php:8 Algo aconteceu. (911) [MyCustomError]
/home/bjori/ex.php:6 Um erro! (112) [InvalidArgumentError]

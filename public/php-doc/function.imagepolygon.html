imagepolygon

(PHP 4, PHP 5, PHP 7, PHP 8)

imagepolygon — Desenha um poligono

Descrição

imagepolygon(
    resource $image,
    array $points,
    int $num_points,
    int $color
): int

imagepolygon() cria um poligono em uma imagem. points é uma matriz PHP contendo os vértices do poligono, ex. points[0] = x0, points[1] = y0, points[2] = x1, points[3] = y1, etc. num_points é o número total de pontos(vértices).

Exemplo #1 Exemplo imagepolygon()

<?php
// cria uma imagem em branco
$image = imagecreate(400, 300);

// prenche a cor do fundo
$bg = imagecolorallocate($image, 0, 0, 0);

// escolhe uma cor para o poligono
$col_poly = imagecolorallocate($image, 255, 255, 255);

// desenha o poligono
imagepolygon($image,
array (
0, 0,
100, 200,
300, 200
),
3,
$col_poly);

// envia a imagem
header("Content-type: image/png");
imagepng($image);

?>

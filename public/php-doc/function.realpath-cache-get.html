realpath_cache_get

(PHP 5 >= 5.3.2, PHP 7, PHP 8)

realpath_cache_get — Obter entradas de cache do caminho absoluto

Descrição

realpath_cache_get(): array

Obtém o conteúdo do cache do caminho absoluto.

Valor Retornado

Retorna um array com as entradas do caminho absoluto. As chaves são o caminho absoluto, e os valores são arrays de informações dos dados, contendo o caminho absoluto, data de expiração, e outras informações mantidas no cache.

Exemplos

Exemplo #1 realpath_cache_get() exemplo

<?php
var_dump(realpath_cache_get());
?>

O exemplo acima produzirá algo semelhante a:

array(2) {
  ["/test"]=>
  array(4) {
    ["key"]=>
    int(123456789)
    ["is_dir"]=>
    bool(true)
    ["realpath"]=>
    string(5) "/test"
    ["expires"]=>
    int(1260318939)
  }
  ["/test/test.php"]=>
  array(4) {
    ["key"]=>
    int(987654321)
    ["is_dir"]=>
    bool(false)
    ["realpath"]=>
    string(12) "/root/test.php"
    ["expires"]=>
    int(1260318939)
  }
}

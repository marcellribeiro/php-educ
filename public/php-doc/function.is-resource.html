is_resource

(PHP 4, PHP 5, PHP 7, PHP 8)

is_resource — Informa se a variável é um resource

Descrição

is_resource(mixed $var): bool

Verifica se uma dada variável é um resource.

Parâmetros

var

A variável a ser avaliada.

Valor Retornado

Retorna true se var é um resource, false caso contrário.

Exemplos

Exemplo #1 Exemplo da is_resource()

<?php

$db_link = @mysql_connect('localhost', 'mysql_user', 'mysql_pass');
if (!is_resource($db_link)) {
die('Can\'t connect : ' . mysql_error());
}

?>

Notas

Nota:

O is_resource() não é um método de validação de tipo estrito: retornará false se a var for uma variável de recurso que está fechado.

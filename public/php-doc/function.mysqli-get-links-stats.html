mysqli_get_links_stats

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

mysqli_get_links_stats — Retorna informações sobre links abertos e em cache

Descrição

mysqli_get_links_stats(): array

mysqli_get_links_stats() retorna informações sobre links MySQL abertos e em cache.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

mysqli_get_links_stats() retorna um array associativo com três elementos, digitados da seguinte forma:

total

Um int que indica o número total de links abertos em qualquer estado.

active_plinks

Um int representando o número de conexões persistentes ativas.

cached_plinks

Um int representando o número de conexões persistentes inativas.

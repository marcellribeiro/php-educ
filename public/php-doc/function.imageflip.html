imageflip

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

imageflip — Flips an image using a given mode

Descrição

imageflip(GdImage $image, int $mode): bool

Flips the image image using the given mode.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

mode

Flip mode, this can be one of the IMG_FLIP_* constants:

Constant Meaning
IMG_FLIP_HORIZONTAL Flips the image horizontally.
IMG_FLIP_VERTICAL Flips the image vertically.
IMG_FLIP_BOTH Flips the image both horizontally and vertically.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Changelog

Versão Descrição
8.0.0 O parâmetro image agora espera uma instância de GdImage; anteriormente, um resource gd válido era esperado.

Exemplos

Exemplo #1 Flips an image vertically

This example uses the IMG_FLIP_VERTICAL constant.

<?php
// File
$filename = 'phplogo.png';

// Content type
header('Content-type: image/png');

// Load
$im = imagecreatefrompng($filename);

// Flip it vertically
imageflip($im, IMG_FLIP_VERTICAL);

// Output
imagejpeg($im);
imagedestroy($im);
?>

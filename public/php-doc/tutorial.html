Aqui nós iremos mostrar o básico do básico do PHP em um simples e curto tutorial. Este texto fala somente sobre a criação de páginas web dinâmicas com PHP, embora o PHP não seja apenas capaz de criar páginas web. Veja a seção intitulada O que o PHP pode fazer para mais informações.

Fazer páginas com PHP é o mesmo que criar páginas HTML e você pode criá-las e editá-las da mesma maneira que você normalmente cria páginas HTML comuns.

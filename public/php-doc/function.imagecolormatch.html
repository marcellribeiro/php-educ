imagecolormatch

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

imagecolormatch — Faz com que a versão das cores da paleta sejam mais próximas da versão truecolor

Descrição

imagecolormatch(resource $image1, resource $image2): bool
Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

image1 deve ser Truecolor, image2 deve ser uma paleta, e ambas image1 e image2 devem ter o mesmo tamanho.

Retorna true em caso de sucesso ou false em caso de falha.

Veja também imagecreatetruecolor().

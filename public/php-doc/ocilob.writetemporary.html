OCI-Lob::writeTemporary

(No version information available, might only be in Git)

OCI-Lob::writeTemporary — Escreve um large object temporário

Descrição

OCI-Lob::writeTemporary(string $data, int $lob_type = OCI_TEMP_CLOB): bool

Cria um large object temporário e escreve data nele.

Você deve usar OCILob::close quando tiver terminado com esse objeto.

Parâmetros

data

Os dados a escrever.

lob_type

Pode ser um dos seguintes:

  • OCI_TEMP_BLOB é utilizado para cirar BLOBs temporários
  • OCI_TEMP_CLOB é utilizado para criar CLOBs temporários

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

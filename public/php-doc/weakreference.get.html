WeakReference::get

(PHP 7 >= 7.4.0, PHP 8)

WeakReference::get — Obtém um objeto fracamente referenciado

Descrição

public WeakReference::get(): ?object

Obtém um objeto fracamente referenciado. Se o objeto já tiver sido destruído, null é retornado.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna o object referenciado ou null se o objeto tiver sido destruído.

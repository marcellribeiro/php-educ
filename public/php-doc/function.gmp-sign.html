gmp_sign

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_sign — Sinal do número

Descrição

gmp_sign(resource $a): int

Verifica o sinal de um número.

Parâmetros

a

Um objeto GMP, um int ou uma string numérica.

Valor Retornado

Retorna 1 se a é positivo, -1 se a é negativo, e 0 se a for zero.

Exemplos

Exemplo #1 Exemplo da gmp_sign()

<?php
// positivo
echo gmp_sign("500") . "\n";

// negativo
echo gmp_sign("-500") . "\n";

// zero
echo gmp_sign("0") . "\n";
?>

O exemplo acima produzirá:

1
-1
0

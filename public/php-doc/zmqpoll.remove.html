ZMQPoll::remove

(PECL zmq >= 0.5.0)

ZMQPoll::remove — Remove um item da lista de consulta

Descrição

public ZMQPoll::remove(mixed $item): boolean

Remove o item da lista de consulta. O parâmetro item pode ser um objeto ZMQSocket, um recurso de fluxo (stream) ou o id retornado pelo método ZMQPoll::add().

Parâmetros

item

O objeto ZMQSocket, recurso de fluxo (stream) ou a string id do item.

Valor Retornado

Retorna verdadeiro caso o item tenha sido removido e falso caso o objeto com o id informado não existe na lista de pesquisa.

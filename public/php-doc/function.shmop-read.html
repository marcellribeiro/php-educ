shmop_read

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

shmop_read — Lê dados de um bloco de memória compartilhada

Descrição

shmop_read(int $shmid, int $start, int $count): string

shmop_read() irá ler uma string de um bloco de memória compartilhada.

shmop_read() pega 3 parâmetros: shmid, que é o identificador do bloco de memória compartilhada criado por shmop_open(), e o último que começa lendo e contando o número de bytes para ler.

Exemplo #1 Lendo um bloco de memória compartilhada

<?php
$shm_data = shmop_read($shm_id, 0, 50);
?>

Esse exemplo irá ler 50 bytes de um bloco de memória compartilhada e armazenaros dados dentro de $shm_data.

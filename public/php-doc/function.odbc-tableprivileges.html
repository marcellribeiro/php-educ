odbc_tableprivileges

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_tableprivileges — Lista tabelas e os privilégios associados a cada tabela

Descrição

odbc_tableprivileges(
    resource $connection_id,
    string $qualifier,
    string $owner,
    string $name
): resource

Lista as tabelas solicitadas e os privilégios associados a cada tabela. Retorna um identificador de resultado ODBC ou false em caso de falha.

O conjunto dos resultados tem as seguintes colunas:

  • TABLE_QUALIFIER
  • TABLE_OWNER
  • TABLE_NAME
  • GRANTOR
  • GRANTEE
  • PRIVILEGE
  • IS_GRANTABLE

O conjunto dos resultados e ordenado por TABLE_QUALIFIER, TABLE_OWNER e TABLE_NAME.

Os argumentos owner e name aceitam caracteres de busca ('%' para buscar zero ou mais caracteres e '_' para buscar um único caractere).

gmp_jacobi

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_jacobi — Símbolo de Jacobi

Descrição

gmp_jacobi(resource $a, resource $p): int

Computa o » Símbolo de Jacobi de a e p. p deve ser ímpar e positivo.

Parâmetros

a

Um objeto GMP, um int ou uma string numérica.

p

Um objeto GMP, um int ou uma string numérica.

Deve ser ímpar e positivo.

Valor Retornado

Um objeto GMP.

Exemplos

Exemplo #1 Exemplo gmp_jacobi()

<?php
echo gmp_jacobi("1", "3") . "\n";
echo gmp_jacobi("2", "3") . "\n";
?>

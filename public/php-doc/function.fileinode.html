fileinode

(PHP 4, PHP 5, PHP 7, PHP 8)

fileinode — Lê o inode do arquivo

Descrição

fileinode(string $nomedoarquivo): int

Obtém o inode do arquivo.

Parâmetros

filename

Caminho para o arquivo.

Valor Retornado

Retorna o número do inode do arquivo, ou false em caso de um erro.

Notas

Nota: Os resultados desta função são armazenados em cache. Consulte a função clearstatcache() para mais detalhes.

Dica

A partir do PHP 5.0.0, esta função também pode ser usada com alguns wrappers URL. Consulte Protocolos e Wrappers suportados para determinar quais wrappers suportam a família de funções stat().

readline_completion_function

(PHP 4, PHP 5, PHP 7, PHP 8)

readline_completion_function — Registra uma função de completar

Descrição

readline_completion_function(callback $function): bool

Esta função registra uma função de completar. Este é o mesmo tipo de funcionamento que você tem se você pressionar a tecla tab enquanto estiver usando o Bash.

Parâmetros

function

Você deve usar o nome de uma função existente a qual aceita uma linha de comando parcial e retorna uma matriz de possíveis combinações.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

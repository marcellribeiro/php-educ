is_object

(PHP 4, PHP 5, PHP 7, PHP 8)

is_object — Informa se a variável é um objeto

Descrição

is_object(mixed $var): bool

Verifica se uma dada variável é um objeto.

Parâmetros

var

A variável a ser avaliada.

Valor Retornado

Retorna true se var é um object, false senão.

Notas

Nota:

Esta função retornará false se usada em um objeto não serializado onde a definição da classe não está presente (mesmo que gettype() retorne object).

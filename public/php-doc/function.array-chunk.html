array_chunk

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

array_chunk — Divide um array em pedaços

Descrição

array_chunk(array $array, int $length, bool $preserve_keys = false): array

Divide um array em length pedaços. O último pedaço pode conter menos elementos que o parâmetro length.

Parâmetros

array

O array.

length

O tamanho de cada pedaço

preserve_keys

Quando definido para true, chaves serão preservadas. O padrão é false que reindexará os pedaços numericamente

Valor Retornado

Retorna um array multidimensional numericamente indexado, iniciando com 0, com cada dimensão contendo length elementos.

Erros

Se length é menor que 1 ValueError será disparado.

Changelog

Versão Descrição
8.0.0 Se length é menor que 1, ValueError será disparado agora; antes, um erro de nível E_WARNING será disparado ao invez de ValueError, e a função retornará null.

Exemplos

Exemplo #1 Exemplo da função array_chunk()

<?php
$input_array = array('a', 'b', 'c', 'd', 'e');
print_r(array_chunk($input_array, 2));
print_r(array_chunk($input_array, 2, true));
?>

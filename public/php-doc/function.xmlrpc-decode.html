xmlrpc_decode

(PHP 4 >= 4.1.0, PHP 5, PHP 7)

xmlrpc_decode — Decodifica XML para tipos nativos do PHP

Descrição

xmlrpc_decode(string $xml, string $encoding = ?): mixed
Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome e documentação poderão mudar sem aviso prévio em futuras versões do PHP. Use por sua conta e risco.

Parâmetros

xml

Resposta XML retornada por método XMLRPC.

encoding

Codificação da entrada suportada por iconv (padrão é "iso-8859-1").

Valor Retornado

Retorna ou um array, ou um inteiro, ou uma string, ou um booleano de acordo com a resposta retornada via XMLRPC.

imagegd2

(PHP 4 >= 4.0.7, PHP 5, PHP 7, PHP 8)

imagegd2 — Envia uma imagem GD2

Descrição

imagegd2(
    resource $image,
    string $filename = ?,
    int $chunk_size = ?,
    int $type = ?
): bool

imagegd2() envia uma imagem GD2 para o browser ou um arquivo.

O parâmetro opcional type pode ser IMG_GD2_RAW ou IMG_GD2_COMPRESSED. O padrão é IMG_GD2_RAW.

Nota:

Os parâmetros opcionais chunk_size e type tornaram-se disponíveis no PHP 4.3.2.

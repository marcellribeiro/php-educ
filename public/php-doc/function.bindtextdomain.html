bindtextdomain

(PHP 4, PHP 5, PHP 7, PHP 8)

bindtextdomain — Configura o caminho para um domínio

Descrição

bindtextdomain(string $domain, string $directory): string

A função bindtextdomain() configura um caminho para um domínio.

Parâmetros

domain

O domínio

directory

O caminho do diretório

Valor Retornado

O caminho completo para o domain que foi definido.

Exemplos

Exemplo #1 Exemplo da bindtextdomain()

<?php

$domain = 'myapp';
echo bindtextdomain($domain, '/usr/share/myapp/locale');

?>

O exemplo acima produzirá:

/usr/share/myapp/locale

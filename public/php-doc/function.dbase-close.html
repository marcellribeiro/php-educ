dbase_close

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_close — Fecha um banco de dados

Descrição

dbase_close(int $dbase_identifier): bool

Fecha o idendificador de banco de dados indicado.

Parâmetros

dbase_identifier

O identificador do banco de dados, retornado por dbase_open() ou dbase_create().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Fechando um banco de dados dBase

<?php

// open in read-only mode
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
// read some data ..

dbase_close($db);
}

?>

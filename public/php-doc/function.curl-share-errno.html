curl_share_errno

(PHP 7 >= 7.1.0, PHP 8)

curl_share_errno — Retorna o último número de erro de curl de compartilhamentor

Descrição

curl_share_errno(CurlShareHandle $share_handle): int

Retorna um número inteiro contendo o último número de erro de curl compartilhado.

Parâmetros

share_handle

Um manipulador cURL de compartilhamento retornado por curl_share_init().

Valor Retornado

Retorna um número inteiro contendo o último número de erro de curl compartilhado ou false em caso de falha.

Changelog

Versão Descrição
8.0.0 A função não retorna mais false em caso de falha.
8.0.0 O parâmetro share_handle agora espera uma instância de CurlShareHandle; anteriormente, um resource era esperado.

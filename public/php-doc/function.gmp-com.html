gmp_com

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_com — Calcula o complemento

Descrição

gmp_com(resource $a): resource

Retorna o complemento de a

Parâmetros

a

Um objeto GMP, um int ou uma string numérica.

Valor Retornado

Retorna o complemento de a, como um número GMP.

Exemplos

Exemplo #1 Exemplo gmp_com()

<?php
$com = gmp_com("1234");
echo gmp_strval($com) . "\n";
?>

O exemplo acima produzirá:

-1235

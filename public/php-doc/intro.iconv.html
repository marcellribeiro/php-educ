Introdução

Este módulo contém uma interface para o recurso de conversão do conjunto de caracteres iconv. Com este módulo, você pode transformar uma string representada por um conjunto de caracteres local em outra representada por outro conjunto de caracteres, que pode ser o conjunto de caracteres Unicode. Os conjuntos de caracteres suportados dependem da implementação iconv do seu sistema. Observe que a função iconv em alguns sistemas pode não funcionar conforme o esperado. Nesse caso, seria uma boa ideia instalar a biblioteca » GNU libiconv. Provavelmente acabará com resultados mais consistentes.

Esta extensão vem com várias funções utilitárias que ajudam você a escrever scripts multilíngues. Vamos dar uma olhada nas seções a seguir para explorar os novos recursos.

date_parse_from_format

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

date_parse_from_format — Retorna informações sobre uma data informada de acordo com um formato

Descrição

date_parse_from_format(string $format, string $date): array

Retorna um array associativo com informações da data informada.

Parâmetros

format

Formato aceito pelo método DateTime::createFromFormat().

date

String que representa a data.

Valor Retornado

Retorna um array associativo com informações da data informada.

Exemplos

Exemplo #1 Exemplo da função date_parse_from_format()

<?php
$date = "6.1.2009 13:00+01:00";
print_r(date_parse_from_format("j.n.Y H:iP", $date));
?>

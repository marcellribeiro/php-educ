odbc_close

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_close — Fecha uma conexão ODBC

Descrição

odbc_close(resource $connection_id): void

Fecha a conexão com o servidor de banco de dados.

Parâmetros

connection_id

O identificador de conexão.

Valor Retornado

Nenhum valor é retornado.

Notas

Nota:

Esta função irá falhar se houverem transações abertas nesta conexão. A conexão permanecerá aberta neste caso.

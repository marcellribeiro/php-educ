zip_entry_read

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_entry_read — Lê de um arquivo aberto

Descrição

zip_entry_read(resource $zip_entry, int $length = 1024): string

Lê de um arquivo aberto.

Parâmetros

zip_entry

Um arquivo retornado pela zip_read().

length

O número de bytes para retornar.

Nota:

Este deve ser o tamanho não comprimido que você deseja ler.

Valor Retornado

Retorna a informação lida, string vazia no fim de um arquivo, ou false em caso de erro.

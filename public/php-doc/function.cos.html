cos

(PHP 4, PHP 5, PHP 7, PHP 8)

cos — Coseno

Descrição

cos(float $arg): float

cos() retorna o coseno de arg. O parâmetro arg deve estar em radianos.

Parâmetros

arg

Um ângulo em radianos

Valor Retornado

O cosseno de arg

Exemplos

Exemplo #1 Exemplo da cos()

<?php

echo cos(M_PI); // -1

?>

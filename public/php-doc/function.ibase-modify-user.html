ibase_modify_user

(PHP 5, PHP 7 < 7.4.0)

ibase_modify_user — Modifica um usuário para um banco de dados seguro (somente para IB6 ou posterior)

Descrição

ibase_modify_user(
    resource $service_handle,
    string $user_name,
    string $password,
    string $first_name = ?,
    string $middle_name = ?,
    string $last_name = ?
): bool
Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

O PHP 4 usa server, dba_user_name e dba_user_password ao invés do parâmetro service_handle.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

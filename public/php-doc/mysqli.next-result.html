mysqli::next_result

mysqli_next_result

(PHP 5, PHP 7, PHP 8)

mysqli::next_result -- mysqli_next_result — Prepara o próximo resultado de multi_query

Descrição

mysqli::next_result(): bool
mysqli_next_result(mysqli $link): bool

Prepara o próximo conjunto de resultado de uma chamada anterior para mysqli_multi_query() que pode ser recuperado por mysqli_store_result() ou mysqli_use_result().

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Veja mysqli_multi_query().

  • mysqli_more_results() - Verifica se há mais algum resultado de uma multi query

  • mysqli_use_result() - Iniciar uma recuperação de conjunto de resultados

mysqli::refresh

mysqli_refresh

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

mysqli::refresh -- mysqli_refresh — Atualiza

Descrição

Estilo orientado a objetos

public mysqli::refresh(int $flags): bool

Estilo procedural

mysqli_refresh(mysqli $mysql, int $flags): bool

Libera tabelas ou caches ou redefine as informações do servidor de replicação.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

flags

As opções para atualizar, usando as constantes MYSQLI_REFRESH_* conforme documentado na documentação das MySQLi constantes.

Veja também a documentação oficial do » MySQL Refresh.

Valor Retornado

true se a atualização foi bem-sucedida, caso contrário, false

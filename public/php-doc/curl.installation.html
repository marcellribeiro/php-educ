Instalação

Para usar o suporte a cURL no PHP, é necessário compilar o PHP com a opção --with-curl[=DIR] onde DIR é o diretório que contém os diretórios lib e include. No diretório include deve haver uma pasta chamada curl que deve conter os arquivos easy.h e curl.h. Deve haver um arquivo chamado libcurl.a localizado na pasta lib.

Nota: Observação para os usuários de Win32
Para ativar este módulo em ambiente Windows, libeay32.dll e ssleay32.dll ou a partir do OpenSSL 1.1 libcrypto-*.dll e libssl-*.dll, precisam estar presentes no seu PATH. A libssh2.dll tambem precisa estar no seu PATH. Você não precisa da libcurl.dll do site do cURL.

octdec

(PHP 4, PHP 5, PHP 7, PHP 8)

octdec — Octal para decimal

Descrição

octdec(string $octal_string): number

Retorna o decimal equivalente do número octal representado pelo argumento octal_string.

Parâmetros

octal_string

A string octal a ser convertida

Valor Retornado

A representação decimal de octal_string

Exemplos

Exemplo #1 Exemplo da octdec()

<?php
echo octdec('77') . "\n";
echo octdec(decoct(45));
?>

O exemplo acima produzirá:

63
45

Notas

Nota:

A função pode converter números que são muito grandes para caber no tipo integer da plataforma, e nesse caso os valores são retornados como float.

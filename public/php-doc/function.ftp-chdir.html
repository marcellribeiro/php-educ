ftp_chdir

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_chdir — Muda o diretório atual em um servidor FTP

Descrição

ftp_chdir(resource $ftp_stream, string $directory): bool

Muda o diretório atual para o especificado.

Parâmetros

ftp_stream

O identificador da conexão FTP.

directory

O diretório de destino.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha. Se houver falha ao mudar o diretório, o PHP irá também enviar um warning.

Exemplos

Exemplo #1 Exemplo ftp_chdir()

<?php

// estabelece a conexão
$conn_id = ftp_connect($ftp_server);

// login com o nome de usuário e senha
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// confere a conexão
if ((!$conn_id) || (!$login_result)) {
die("A conexão FTP falhou!");
}

echo "Diretório atual: " . ftp_pwd($conn_id) . "\n";

// tenta mudar para algumDiretorio
if (ftp_chdir($conn_id, "algumDiretorio")) {
echo "O diretório atual agora é: " . ftp_pwd($conn_id) . "\n";
} else {
echo "Não foi possível mudar o diretório\n";
}
// fecha a conexão
ftp_close($conn_id);
?>

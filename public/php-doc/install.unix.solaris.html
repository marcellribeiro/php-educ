Dicas de instalação específicas para o Solaris

Essa seção contêm notas e dicas específicas para a instalação do PHP em sistemas Solaris.

Software Necessário

Instalações do Solaris frequentemente não possuem compiladores C e suas ferramentas relacionadas. Leia esse FAQ para informação sobre porque usar versões GNU para algumas dessas ferramentas é necessário.

Para descompactar uma distribuição PHP você precisará de:

  • tar
  • gzip or
  • bzip2

Para compilar o PHP você precisará de:

  • gcc (recomendado, outros compiladores C podem funcionar)
  • make
  • GNU sed

Para compilar extensões extras ou hackear o código do PHP você também precisará de:

  • re2c
  • bison
  • m4
  • autoconf
  • automake
Além você pode precisar instalar (e talvez até compilar) softwares específicos à sua configuração, como o Oracle ou MySQL.

Utilizando pacotes

Você pode simplificar o processo de instalação no Solaris ao utilizar pkgadd para instalar os componentes mais comuns. O Image Packaging System (IPS) para Solaris 11 Express também contém os componentes mais comuns utilizando-se o comando pkg.

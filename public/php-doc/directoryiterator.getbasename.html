DirectoryIterator::getBasename

(PHP 5 >= 5.2.2, PHP 7, PHP 8)

DirectoryIterator::getBasename — Retorna o nome do item corrente do DirectoryIterator.

Descrição

public DirectoryIterator::getBasename( string $sufixo = ?): string

Retorna o nome do item corrente do DirectoryIterator.

Parâmetros

sufixo

Se o nome base terminar no sufixo, ele é retirado.

Valor Retornado

O nome do item corrente do DirectoryIterator.

Exemplos

Exemplo #1 Exemplo do DirectoryIterator::getBasename()

Esse exemplo irá listar todo o nome base e o nome base com o sufixo .jpg removido dos nomes de arquivo do diretório o qual o script irá rodar.

<?php
$dir = new DirectoryIterator(dirname(__FILE__));
foreach ($dir as $fileinfo) {
if ($fileinfo->isFile()) {
echo $fileinfo->getBasename() . "\n";
echo $fileinfo->getBasename('.jpg') . "\n";
}
}
?>

O exemplo acima produzirá algo semelhante a:

apple.jpg
apple
banana.jpg
banana
index.php
index.php
pear.jpg
pear

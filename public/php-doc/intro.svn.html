Introdução

This extension implements PHP bindings for » Subversion (SVN), a version control system, allowing PHP scripts to communicate with SVN repositories and working copies without direct command line calls to the svn executable.

Aviso

Esta extensão é EXPERIMENTAL. O comportamento desta extensão, incluindo os nomes de suas funções e qualquer outra documentação sobre esta extensão, poderá mudar sem aviso prévio em futuras versões do PHP. Use por sua conta e risco.

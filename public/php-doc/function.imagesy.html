imagesy

(PHP 4, PHP 5, PHP 7, PHP 8)

imagesy — Obtém a altura da imagem

Descrição

imagesy(resource $image): int

Retorna a altura do dado resource image.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

Valor Retornado

Retorna a altura da image ou false em erro.

Exemplos

Exemplo #1 Usando imagesy()

<?php

// create a 300*200 image
$img = imagecreatetruecolor(300, 200);

echo imagesy($img); // 200

?>

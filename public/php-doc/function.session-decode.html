session_decode

(PHP 4, PHP 5, PHP 7, PHP 8)

session_decode — Decodifica dados de sessão de uma sessão codificada em formato string

Descrição

session_decode(string $data): bool

session_decode() decodifica os dado de sessão serializados fornecidos em $data e preenche a super global $_SESSION com o resultado.

Por padrão, o método de deserialização usado é interno do PHP e não é o mesmo que unserialize(). O método de deserialização pode ser configurado usando session.serialize_handler.

Parâmetros

data

Os dados codificados para serem armazenados.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

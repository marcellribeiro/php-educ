imagecreatefromxbm

(PHP 4 >= 4.0.1, PHP 5, PHP 7, PHP 8)

imagecreatefromxbm — Cria uma nova imagem a a partir de um arquivo ou URL

Descrição

imagecreatefromxbm(string $filename): resource

imagecreatefromxbm() retorna um identificador de imagem representando a imagem obtida através do nome de arquivo dado.

Dica

Uma URL pode ser usada como um nome de arquivo com esta função se os wrappers fopen estiverem habilitados. Consulte a função fopen() para mais detalhes sobre como especificar o nome do arquivo. Consulte Protocolos e Wrappers suportados para informações sobre as capacidades de cada wrapper, notas de uso e informações sobre quaisquer variáveis predefinidas que eles possam fornecer.

Parâmetros

filename

Caminho para imagem XBM

Valor Retornado

Retorna um resource identificador da imagem em sucesso, false em erros.

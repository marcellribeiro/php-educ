var_dump

(PHP 4, PHP 5, PHP 7, PHP 8)

var_dump — Mostra informações sobre a variável

Descrição

var_dump(mixed $expression, mixed $expression = ?, $... = ?): void

Esta função mostrará uma representação estruturada sobre uma ou mais expressões, incluindo o tipo e o valor. Arrays e objetos são explorados recursivamente com valores identados na estrutura mostrada.

No PHP 5 todas propriedades public, private e protected do objeto são retornados na saída.

Dica

Assim como qualquer coisa que envia seu resultado diretamente para o navegador, as funções de controle de saída podem ser usadas para capturar a saída desta função e salvá-la em uma string (por exemplo).

Parâmetros

expression

A variável que você quer exportar.

Valor Retornado

Nenhum valor é retornado.

Exemplos

Exemplo #1 Exemplo da var_dump()

<?php
$a = array (1, 2, array ("a", "b", "c"));
var_dump ($a);
?>

O exemplo acima produzirá:

/* mostrará:
array(3) {
  [0]=>
  int(1)
  [1]=>
  int(2)
  [2]=>
  array(3) {
    [0]=>
    string(1) "a"
    [1]=>
    string(1) "b"
    [2]=>
    string(1) "c"
  }
}
<?php

$b = 3.1;
$c = true;

?>

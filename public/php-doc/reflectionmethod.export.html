ReflectionMethod::export

(PHP 5, PHP 7)

ReflectionMethod::export — Export a reflection method

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 7.4.0 e foi REMOVIDA a partir do PHP 8.0.0. O uso desta função é altamente desaconselhado.

Descrição

public static ReflectionMethod::export(string $class, string $name, bool $return = false): string

Exports a ReflectionMethod.

Parâmetros

class

The class name.

name

The name of the method.

return

Definir como true retornará a exportação, em vez de emiti-la. Definir como false (o padrão) fará o oposto.

Valor Retornado

Se o parâmetro return for definido como true, então a exportação será retornada como uma string, caso contrário, null será retornado.

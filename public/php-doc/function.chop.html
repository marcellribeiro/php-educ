chop

(PHP 4, PHP 5, PHP 7, PHP 8)

chop — Sinônimo de rtrim()

Descrição

Esta função é um apelido para: rtrim().

Notas

Nota:

chop() é diferente da função do Perl chop(), que remove o último caractere da string.

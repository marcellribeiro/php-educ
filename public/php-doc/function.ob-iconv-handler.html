ob_iconv_handler

(PHP 4 >= 4.0.5, PHP 5, PHP 7, PHP 8)

ob_iconv_handler — Converte codificação de caracteres como manipulador de buffer de saída

Descrição

ob_iconv_handler(string $contents, int $status): string

Converte a string codificada em internal_encoding em output_encoding.

internal_encoding e output_encoding devem ser definidos no arquivo php.ini ou em iconv_set_encoding().

Parâmetros

Consulte ob_start() para obter informações sobre os parâmetros desse manipulador.

Valor Retornado

Consulte ob_start() para obter informações sobre os valores de retorno desse manipulador.

Exemplos

Exemplo #1 Exemplo ob_iconv_handler():

<?php
iconv_set_encoding("internal_encoding", "UTF-8");
iconv_set_encoding("output_encoding", "ISO-8859-1");
ob_start("ob_iconv_handler"); // start output buffering
?>

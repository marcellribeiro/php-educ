ZMQPoll::getLastErrors

(PECL zmq >= 0.5.0)

ZMQPoll::getLastErrors — Obtém erros da consulta

Descrição

public ZMQPoll::getLastErrors(): array

Retorna os ids dos objetos que tiveram erro na última consulta.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna um array contendo os ids dos itens que tiveram erro na última consulta. Um arrau vazio é retornado se não houveram erros.

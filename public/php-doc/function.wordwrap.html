wordwrap

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

wordwrap — Quebra uma string em um dado número de caracteres

Descrição

wordwrap(
    string $str,
    int $width = 75,
    string $break = "\n",
    bool $cut = false
): string

Quebra uma string em um dado número de caracteres usando um caractere de quebra de string.

Parâmetros

str

A string de entrada.

width

A coluna da largura. O padrão é 75.

break

A linha é quebrada usando o parâmetro opcional break. O padrão é '\n'.

cut

Se cut é setado para true, a string é sempre quebrada na largura especificada ou antes. Então se você tem uma palavra que é mais larga que a largura dada, ela é sempre quebrada. (Veja segundo exemplo).

Valor Retornado

Retorna a string dada quebrada na largura especificada.

Exemplos

Exemplo #1 Exemplo de wordwrap()

<?php
$text = "The quick brown fox jumped over the lazy dog.";
$newtext = wordwrap($text, 20, "<br />\n");

echo $newtext;
?>

O exemplo acima produzirá:

The quick brown fox<br />
jumped over the lazy<br />
dog.

Exemplo #2 Exemplo de wordwrap()

<?php
$text = "A very long woooooooooooord.";
$newtext = wordwrap($text, 8, "\n", true);

echo "$newtext\n";
?>

O exemplo acima produzirá:

A very
long
wooooooo
ooooord.

mysqli::$error_list

mysqli_error_list

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

mysqli::$error_list -- mysqli_error_list — Retorna uma lista de erros do último comando executado

Descrição

Estilo orientado a objetos

array $mysqli->error_list;

Estilo procedural

mysqli_error_list(mysqli $mysql): array

Retorna uma matriz de erros para a chamada de função MySQLi mais recente que pode ser bem-sucedida ou falhar.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

Valor Retornado

Uma lista de erros, cada um como um array associativo contendo errno, error e sqlstate.

Exemplos

Exemplo #1 $mysqli->error_list exemplo

Estilo orientado a objetos

<?php
$mysqli = new mysqli("localhost", "nobody", "");

/* verifica a conexão */
if (mysqli_connect_errno()) {
printf("Connect failed: %s\n", mysqli_connect_error());
exit();
}

if (!$mysqli->query("SET a=1")) {
print_r($mysqli->error_list);
}

/* fecha a conexão */
$mysqli->close();
?>

Estilo procedural

<?php
$link = mysqli_connect("localhost", "my_user", "my_password", "world");

/* verifica a conexão */
if (mysqli_connect_errno()) {
printf("Connect failed: %s\n", mysqli_connect_error());
exit();
}

if (!mysqli_query($link, "SET a=1")) {
print_r(mysqli_error_list($link));
}

/* fecha a conexão */
mysqli_close($link);
?>

Os exemplos acima produzirão:

Array
(
    [0] => Array
        (
            [errno] => 1193
            [sqlstate] => HY000
            [error] => Unknown system variable 'a'
        )

)

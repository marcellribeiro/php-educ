mysqli::get_client_info

mysqli_get_client_info

(PHP 5, PHP 7, PHP 8)

mysqli::get_client_info -- mysqli_get_client_info — Retorna a versão do cliente MySQL como uma string

Descrição

mysqli::get_client_info(): string
mysqli_get_client_info(): string

A função mysqli_get_client_info() é usada para retornar uma string representando a versão de cliente usada na extensão MySQLi.

Valor Retornado

Uma string que representa a versão da biblioteca de cliente do MySQL.

Exemplos

Exemplo #1 mysqli_get_client_info

<?php

/* We don't need a connection to determine
the version of mysql client library */

printf("Client library version: %s\n", mysqli_get_client_info());
?>

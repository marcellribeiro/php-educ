wddx_packet_end

(PHP 4, PHP 5, PHP 7)

wddx_packet_end — Finaliza um pacote WDDX com o ID especificado

Descrição

wddx_packet_end(resource $packet_id): string

Finaliza e retorna um pacote WDDX escolhido.

Parâmetros

packet_id

Um pacote WDDX, retornado por wddx_packet_start().

Valor Retornado

Retorna a string com o pacote WDDX.

SolrQuery::getMltMaxNumTokens

(PECL solr >= 0.9.2)

SolrQuery::getMltMaxNumTokens — Retorna o número máximo de tokens para analisar em cada campo do documento que não é armazenado com suporte a TermVector

Descrição

public SolrQuery::getMltMaxNumTokens(): int

Retorna o número máximo de tokens para analisar em cada campo do documento que não é armazenado com suporte a TermVector

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna um inteiro em caso de sucesso e null se não for definido.

mysqli::ssl_set

mysqli_ssl_set

(PHP 5, PHP 7, PHP 8)

mysqli::ssl_set -- mysqli_ssl_set — Usado para estabelecer conexões seguras usando SSL

Descrição

Estilo orientado a objetos

public mysqli::ssl_set(
    ?string $key,
    ?string $certificate,
    ?string $ca_certificate,
    ?string $ca_path,
    ?string $cipher_algos
): bool

Estilo procedural

mysqli_ssl_set(
    mysqli $mysql,
    ?string $key,
    ?string $certificate,
    ?string $ca_certificate,
    ?string $ca_path,
    ?string $cipher_algos
): bool

Usado para estabelecer conexões seguras usando SSL. Deve ser chamado antes de mysqli_real_connect(). Esta função não faz nada, a menos que o suporte OpenSSL esteja ativado.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

key

O nome do caminho para o arquivo de chave.

certificate

O nome do caminho para o arquivo de certificado.

ca_certificate

O nome do caminho para o arquivo de autoridade de certificação.

ca_path

O nome do caminho para um diretório que contém certificados SSL CA confiáveis no formato PEM.

cipher_algos

Uma lista de cifras permitidas a serem usadas para criptografia SSL.

Valor Retornado

Esta função sempre retorna o valor true. Se a configuração do SSL estiver incorreta, mysqli_real_connect() retornará um erro quando você tentar se conectar.

enchant_dict_get_error

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_dict_get_error — Returns the last error of the current spelling-session

Descrição

enchant_dict_get_error(EnchantDictionary $dictionary): string|false

Returns the last error of the current spelling-session

Parâmetros

dictionary

Um dicionário Enchant retornado por enchant_broker_request_dict() ou enchant_broker_request_pwl_dict().

Valor Retornado

Returns the error message as string or false if no error occurred.

Changelog

Versão Descrição
8.0.0 O parâmetro dictionary agora espera uma instância de EnchantDictionary; anteriormente, um resource era esperado.

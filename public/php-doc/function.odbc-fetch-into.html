odbc_fetch_into

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_fetch_into — Obtém uma linha do conjunto de resultados como uma matriz

Descrição

odbc_fetch_into(resource $result_id, array &$result_array, int $rownumber = ?): int
odbc_fetch_into(resource $result_id, int $rownumber = ?, array &$result_array): bool

Retorna o número de colunas no resultado; false em caso de erro. result_array deve ser passado por referencia, mas pode ser de qualquer tipo já que vai ser convertido para array. A matriz irá conter os valores das colunas com o índice começando em 0.

A partir do PHP 4.0.5 o parâmetro result_array não precisa mais ser passado por referencia.

A partir do PHP 4.0.6 o parâmetro rownumber não pode ser passado como uma constante, mas sim como uma variável.

A partir do PHP 4.2.0 o parâmetro result_array e rownumber foram trocados. Isto permite a rownumber poder ser uma constante novamente. Esta mudança também será a última nesta função.

Exemplo #1 Exemplo odbc_fetch_into() pré 4.0.6

<?php
$rc = odbc_fetch_into($res_id, $my_array);
?>

ou

<?php
$rc = odbc_fetch_into($res_id, $row, $my_array);

$rc = odbc_fetch_into($res_id, 1, $my_array);
?>

Exemplo #2 Exemplo odbc_fetch_into() 4.0.6

<?php
$rc = odbc_fetch_into($res_id, $my_array);
?>

ou

<?php
$row = 1;
$rc = odbc_fetch_into($res_id, $row, $my_array);
?>

Exemplo #3 Exemplo odbc_fetch_into() 4.2.0

<?php
$rc = odbc_fetch_into($res_id, $my_array);
?>

ou

<?php
$rc = odbc_fetch_into($res_id,$my_array, 2);
?>

curl_pause

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_pause — Pausa e despausa uma conexão

Descrição

curl_pause(resource $ch, int $bitmask): int

Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

Parâmetros

handle

Um manipulador cURL retornado por curl_init().

bitmask

Uma das constantes CURLPAUSE_*.

Valor Retornado

Retorna um código de erro (CURLE_OK se não houver erro).

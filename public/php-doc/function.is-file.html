is_file

(PHP 4, PHP 5, PHP 7, PHP 8)

is_file — Informa se o arquivo é um arquivo comum

Descrição

is_file(string $filename): bool

Informa se o arquivo é um arquivo comum.

Parâmetros

filename

Caminho para o arquivo.

Valor Retornado

Retorna true se o arquivo existe e é um arquivo comum, false caso contrário.

Nota: Como o tipo inteiro do PHP possui sinal e muitas plataformas usam números inteiros de 32 bits, algumas funções do sistema de arquivos podem retornar resultados inesperados para arquivos maiores que 2GB.

Exemplos

Exemplo #1 Exemplo da is_file()

<?php
var_dump(is_file('a_file.txt')) . "\n";
var_dump(is_file('/usr/bin/')) . "\n";
?>

O exemplo acima produzirá:

bool(true)
bool(false)

Erros

Em caso de falha, um E_WARNING será emitido.

Notas

Nota: Os resultados desta função são armazenados em cache. Consulte a função clearstatcache() para mais detalhes.

Dica

A partir do PHP 5.0.0, esta função também pode ser usada com alguns wrappers URL. Consulte Protocolos e Wrappers suportados para determinar quais wrappers suportam a família de funções stat().

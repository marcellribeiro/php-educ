mysqli_sql_exception::getSqlState

(PHP 8 >= 8.1.2)

mysqli_sql_exception::getSqlState — Retorna o código de erro SQLSTATE

Descrição

public mysqli_sql_exception::getSqlState(): string

Retorna uma string contendo o código de erro SQLSTATE para o último erro. O código de erro consiste em cinco caracteres. Os valores são especificados por ANSI SQL e ODBC. Para obter uma lista de valores possíveis, consulte » http://dev.mysql.com/doc/mysql/en/error-handling.html.

Nota:

Observe que nem todos os erros do MySQL ainda estão mapeados para SQLSTATE. O valor HY000 (erro geral) é usado para erros não mapeados.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna uma string contendo o código de erro SQLSTATE para o último erro. O código de erro consiste em cinco caracteres.

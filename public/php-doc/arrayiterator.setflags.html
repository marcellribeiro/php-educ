ArrayIterator::setFlags

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

ArrayIterator::setFlags — Define as flags que irão alterar o comportamento

Descrição

public ArrayIterator::setFlags(string $flags): void

Define as flags que alteram o comportamento do ArrayIterator

Parâmetros

flags

O novo comportamento do ArrayIterator. Pode ser utilizado bitmask ou constantes. O uso de constantes é fortemente encorajado para garantir compatibilidade com versões futuras.

As flags disponíveis estão listadas abaixo. O atual significado das flags estão descritas nas constantes pré-definidas.

Flags do comportamento do ArrayIterator
value constant
1 ArrayIterator::STD_PROP_LIST
2 ArrayIterator::ARRAY_AS_PROPS

Valor Retornado

Nenhum valor é retornado.

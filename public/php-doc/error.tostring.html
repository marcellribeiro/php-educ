Error::__toString

(PHP 7, PHP 8)

Error::__toString — Representação em string do erro

Descrição

public Error::__toString(): string

Retorna a representação em string do erro.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Iterando e imprimindo a trilha de erro.

Exemplos

Exemplo #1 Exemplo do método Error::__toString()

<?php
try {
throw new Error("Mensagem de erro.");
} catch(Error $e) {
echo $e;
}
?>

O exemplo acima produzirá algo semelhante a:

Error: Mensagem de erro. in /home/bjori/tmp/ex.php:3
Stack trace:
#0 {main}

defined

(PHP 4, PHP 5, PHP 7, PHP 8)

defined — Confere se uma constante existe

Descrição

defined(string $name): bool

Verifica se uma dada constante existe e é definida.

Nota:

Se você quer ver se uma variável existe, use isset() como defined() aplica-se somente para constants. Se você quer ver se uma função existe, use function_exists().

Parâmetros

name

O nome da constante.

Valor Retornado

Retorna true se o nome da constante dado em name foi definida, false caso contrário.

Exemplos

Exemplo #1 Conferindo Constantes

<?php
// Note que deve estar entre apostrofros, este
// exemplo esta conferindo se existe uma constante com o nome de CONSTANT
if (defined('CONSTANT')){
echo CONSTANT;
}
?>

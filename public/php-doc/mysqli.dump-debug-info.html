mysqli::dump_debug_info

mysqli_dump_debug_info

(PHP 5, PHP 7, PHP 8)

mysqli::dump_debug_info -- mysqli_dump_debug_info — Descarrega informação de debug no log

Descrição

Estilo orientado a objeto (metodo):

mysqli::dump_debug_info(): bool

Estilo de procedimento:

mysqli_dump_debug_info(mysqli $link): bool

Esta função foi criada para ser executada por um usuário com privilégio SUPER e é usada para descaregar informações no log para o servidor MySQL relacionado a conexão especificada pelo parâmetro link.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

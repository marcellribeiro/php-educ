mysql_field_seek

(PHP 4, PHP 5)

mysql_field_seek — Define o ponteiro do resultado para o índice de campo especificado

Aviso

Esta extensão tornou-se obsoleta a partir do PHP 5.5.0 e foi removida no PHP 7.0.0. Em vez disso, as extensões MySQLi ou PDO_MySQL devem ser usadas. Veja também o guia MySQL: escolhendo uma API. Alternativas a esta função incluem:

  • mysqli_field_seek()
  • PDOStatement::fetch() usando os parâmetros cursor_orientation e offset

Descrição

mysql_field_seek(resource $result, int $field_offset): bool

Move para o índice do campo especificado. Se a próxima chamada a mysql_fetch_field() não incluir um índice, o índice de campo especificado em mysql_field_seek() será retornado.

Parâmetros

result

O resource do resultado que está sendo avaliado. Este resultado vem de uma chamada a mysql_query().

field_offset

O deslocamento do campo. field_offset começa em 0. Se field_offset não existir, um erro de nível E_WARNING também será emitido.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

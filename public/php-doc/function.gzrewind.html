gzrewind

(PHP 4, PHP 5, PHP 7, PHP 8)

gzrewind — Retorna ao início a posição de um ponteiro para um arquivo-gz

Descrição

gzrewind(resource $zp): bool

Define o indicador de posição para o dado arquivo gz para o início do arquivo.

Parâmetros

zp

O ponteiro do arquivo gz. Ele deve ser válido e deve apontar para um arquivo aberto corretamente com gzopen().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

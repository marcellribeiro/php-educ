convert_uudecode

(PHP 5, PHP 7, PHP 8)

convert_uudecode — Decodifica uma string codificada com uuencode

Descrição

convert_uudecode(string $data): string

convert_uudecode() decodifica uma string codificada com uuencode.

Parâmetros

data

A informação codificada com uuencode.

Valor Retornado

Retorna a informação decodificada como uma string.

Exemplos

Exemplo #1 Exemplo da convert_uudecode()

<?php
/* Can you imagine what this will print? :) */
echo convert_uudecode("+22!L;W9E(%!(4\"$`\n`");
?>

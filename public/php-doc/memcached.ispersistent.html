Memcached::isPersistent

(PECL memcached >= 2.0.0)

Memcached::isPersistent — Verifica se uma conexão permanente com o memcache está sendo usada

Descrição

public Memcached::isPersistent(): bool

Memcached::isPersistent() verifica se as conexões com os servidores memcache são conexões persistentes.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna true se a instância do Memcache usar uma conexão persistente, false caso contrário.

ibase_blob_import

(PHP 5, PHP 7 < 7.4.0)

ibase_blob_import — Cria um blob, copia um arquivo para ele e o fecha

Descrição

ibase_blob_import(resource $link_identifier, resource $file_handle): string
ibase_blob_import(resource $file_handle): string

Este função cria um BLOB, lê um arquivo inteiro para dentro dele, fecha, e retorna o id do BLOB.

Parâmetros

link_identifier

Um link identificador InterBase. Se omitido, o último link aberto é usado.

file_handle

O manipulador de arquivo é um manipulador retornado por fopen().

Valor Retornado

Retorna o id do BLOB em sucesso, ou false em erro.

Exemplos

Exemplo #1 Exemplo da ibase_blob_import()

<?php
$dbh = ibase_connect($host, $username, $password);
$filename = '/tmp/bar';

$fd = fopen($filename, 'r');
if ($fd) {

$blob = ibase_blob_import($dbh, $fd);
fclose($fd);

if (!is_string($blob)) {
// import failed
} else {
$query = "INSERT INTO foo (name, data) VALUES ('$filename', ?)";
$prepared = ibase_prepare($dbh, $query);
if (!ibase_execute($prepared, $blob)) {
// record insertion failed
}
}
} else {
// unable to open the data file
}
?>

ZMQSocket::sendmulti

(PECL zmq >= 0.8.0)

ZMQSocket::sendmulti — Envia uma mensagem de múltiplas partes

Descrição

public ZMQSocket::sendmulti(array $message, int $mode = 0): ZMQSocket

Envia uma mensagem com várias partes usando o socket. A operação pode bloquear, exceto caso ZMQ::MODE_NOBLOCK seja usada.

Parâmetros

message

A mensagem a ser enviada - um vetor (array) de strings

mode

Passe indicadores mode para receber mensagens com múltiplas partes ou operações não bloqueantes. Consulte as constantes ZMQ::MODE_*.

Valor Retornado

Retorna o objeto atual. Lança uma exceção ZMQSocketException caso haja erro. Se a constante ZMQ::MODE_NOBLOCK for usada e a operação bloqueie, bool falso será retornado.

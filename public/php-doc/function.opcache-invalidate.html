opcache_invalidate

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL ZendOpcache >= 7.0.0)

opcache_invalidate — Invalida um script em cache

Descrição

opcache_invalidate(string $filename, bool $force = false): bool

Esta função invalida um determinado script do cache opcode. Se force não estiver definido ou for false, o script só será invalidado se o horário de modificação do script for mais recente que os opcodes armazenados em cache. Esta função invalida apenas o cache na memória e não o cache do arquivo.

Parâmetros

filename

O caminho para o script que está sendo invalidado.

force

Se definido como true, o script será invalidado independentemente de a invalidação ser necessária.

Valor Retornado

Retorna true se o cache do opcode para filename foi invalidado ou se não havia nada para invalidar, ou false se o cache do opcode está desabilitado.

curl_share_strerror

(PHP 7 >= 7.1.0, PHP 8)

curl_share_strerror — Retorna uma string descrevendo o código de erro fornecid

Descrição

curl_share_strerror(int $error_code): ?string

Retorna uma mensagem de erro de texto descrevendo o código de erro fornecido.

Parâmetros

error_code

Uma das constantes dos » códigos de erro cURL.

Valor Retornado

Retorna a descrição do erro ou null para código de erro inválido.

ibase_add_user

(PHP 5, PHP 7 < 7.4.0)

ibase_add_user — Adiciona um usuário a um banco de dados seguro (apenas para IB6 ou posterior)

Descrição

ibase_add_user(
    resource $service_handle,
    string $user_name,
    string $password,
    string $first_name = ?,
    string $middle_name = ?,
    string $last_name = ?
): bool

O PHP 4 usa server, dba_user_name e dba_user_password ao invés do parâmetro service_handle.

Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

filegroup

(PHP 4, PHP 5, PHP 7, PHP 8)

filegroup — Lê o grupo do arquivo

Descrição

filegroup(string $filename): int|false

Obtém o grupo do arquivo. O ID do grupo é retornado no formato numérico, use posix_getgrgid() para resolvê-lo para um nome do grupo.

Parâmetros

filename

Caminho do arquivo.

Valor Retornado

Retorna o group ID do arquivo, ou false no caso de um erro. O group ID é retornado em um formato numérico. Use posix_getgrgid() para resolver para o nome do grupo. Em caso de falha, false é retornado.

Erros

Em caso de falha. um E_WARNING é emitido.

Exemplos

Exemplo #1 Descobrindo o grupo de um arquivo

<?php
$filename = 'index.php';
print_r(posix_getgrgid(filegroup($filename)));
?>

Notas

Nota: Os resultados desta função são armazenados em cache. Consulte a função clearstatcache() para mais detalhes.

Dica

A partir do PHP 5.0.0, esta função também pode ser usada com alguns wrappers URL. Consulte Protocolos e Wrappers suportados para determinar quais wrappers suportam a família de funções stat().

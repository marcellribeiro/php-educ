imagecolorstotal

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolorstotal — Retorna o número de cores na paleta de uma imagem

Descrição

imagecolorstotal(resource $image): int

Retorna o número de cores na paleta da imagem.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

Valor Retornado

Retorna o número de cores na especificada paleta da imagem ou 0 para imagens truecolor.

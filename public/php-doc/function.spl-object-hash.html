spl_object_hash

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

spl_object_hash — Retorna uma identificação hash do objeto dado

Descrição

spl_object_hash(object $obj): string

Esta função retorna um identificador único para o objeto. Este identificador por ser usado como uma chave hash para guardar objetos ou para identificar um objeto.

Parâmetros

object

Qualquer objeto.

Valor Retornado

Uma string que é única para cada objeto e é sempre a mesma para o mesmo objeto.

Exemplos

Exemplo #1 Exemplo de spl_object_hash()

<?php
$id = spl_object_hash($object);
$storage[$id] = $object;
?>

imagechar

(PHP 4, PHP 5, PHP 7, PHP 8)

imagechar — Desenha um caractere horizontalmente

Descrição

imagechar(
    resource $image,
    int $font,
    int $x,
    int $y,
    string $c,
    int $color
): int

imagechar() desenha o primeiro caractere de c na imagem identificada por image com o canto superior esquerdo em x,y (em cima a esquerda é 0, 0) com a cor color. Se font for 1, 2, 3, 4 or 5, uma fonte interna é usada (onde os números mais altos correspondem a fonter maiores).

Exemplo #1 Exemplo imagechar()

<?php

$im = imagecreate(100, 100);

$string = 'PHP';

$bg = imagecolorallocate($im, 255, 255, 255);
$black = imagecolorallocate($im, 0, 0, 0);

// mostra um "P" preto no canto superior esquerdo
imagechar($im, 1, 0, 0, $string, $black);

header('Content-type: image/png');
imagepng($im);

?>

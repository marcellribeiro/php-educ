A class CURLStringFile

(PHP 8 >= 8.1.0)

Introdução

CURLStringFile torna possível carregar um arquivo diretamente de uma variável. Isso é semelhante ao CURLFile, mas funciona com o conteúdo do arquivo, não com o nome do arquivo. Esta classe ou CURLFile deve ser usada para carregar o conteúdo do arquivo com CURLOPT_POSTFIELDS.

Sinopse da classe

class CURLStringFile {
/* Propriedades */
public string $data;
public string $postname;
public string $mime;
/* Métodos */
public __construct(string $data, string $postname, string $mime = "application/octet-stream")
}

Propriedades

data

O conteúdo a ser carregado.

postname

O nome do arquivo a ser usado nos dados de upload.

mime

Tipo MIME do arquivo (o padrão é application/octet-stream).

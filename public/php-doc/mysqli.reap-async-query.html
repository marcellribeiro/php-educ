mysqli::reap_async_query

mysqli_reap_async_query

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

mysqli::reap_async_query -- mysqli_reap_async_query — Obtenha o resultado da consulta assíncrona

Descrição

Estilo orientado a objetos

public mysqli::reap_async_query(): mysqli_result|bool

Estilo procedural

mysqli_reap_async_query(mysqli $mysql): mysqli_result|bool

Obtenha o resultado da consulta assíncrona.

Nota:

Disponível apenas com o mysqlnd.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

Valor Retornado

Retorna false em caso de falha. Para consultas bem-sucedidas que produzem um conjunto de resultados, como SELECT, SHOW, DESCRIBE ou EXPLAIN, mysqli_reap_async_query() retornará um objeto mysqli_result. Para outras consultas bem-sucedidas, mysqli_reap_async_query() retornará true.

Erros

Se o relatório de erros da extensão mysqli estiver habilitado (MYSQLI_REPORT_ERROR) e a operação solicitada falhar, um aviso será gerado. Se, além disso, o modo for definido como MYSQLI_REPORT_STRICT, uma exceção mysqli_sql_exception será lançada.

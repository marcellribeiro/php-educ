dbase_pack

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_pack — Apaga permanentemente os registros do banco de dados dBase

Descrição

dbase_pack(int $dbase_identifier): bool

Apaga permanentemente todos os registros marcados para exclusão usando dbase_delete_record().

Parâmetros

dbase_identifier

O identificador do link do banco de dados, retornado por dbase_open() ou dbase_create().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Limpando um banco de dados dBase

<?php

// open in read-write mode
$db = dbase_open('/tmp/test.dbf', 2);

if ($db) {
$record_numbers = dbase_numrecords($db);
for ($i = 1; $i <= $record_numbers; $i++) {
dbase_delete_record($db, $i);
}
// expunge the database
dbase_pack($db);
}

?>

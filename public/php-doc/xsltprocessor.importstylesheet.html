XSLTProcessor::importStylesheet

(PHP 5, PHP 7, PHP 8)

XSLTProcessor::importStylesheet — Importa stylesheet

Descrição

importStylesheet(DOMDocument $stylesheet): void

Este método importa o stylesheet no XSLTProcessor para transformações.

Parâmetros

stylesheet

O importado style sheet como um objeto DOMDocument.

Valor Retornado

Nenhum valor é retornado.

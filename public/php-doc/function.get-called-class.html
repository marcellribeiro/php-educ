get_called_class

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

get_called_class — Retorna o nome da classe "Chamado pelo method estatico"

Descrição

get_called_class(): string

Obtém o nome da classe em que o método estático é chamado.

Valor Retornado

Retorna o nome da classe ou false se chamado de fora de uma classe.

Exemplos

Exemplo #1 Usando get_called_class()

<?php

class foo {
static public function test() {
var_dump(get_called_class());
}
}

class bar extends foo {
}

foo::test();
bar::test();

?>

O exemplo acima produzirá:

string(3) "foo"
string(3) "bar"

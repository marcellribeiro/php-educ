ibase_errcode

(PHP 5, PHP 7 < 7.4.0)

ibase_errcode — Retorna um código de erro

Descrição

ibase_errcode(): int

Retorna o código de erro que resultou da função do InterBase mais recentemente utilizada.

Valor Retornado

Returns the error code as an integer, or false if no error occured.

array_change_key_case

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

array_change_key_case — Modifica a caixa de todas as chaves em um array

Descrição

array_change_key_case(array $array, int $case = CASE_LOWER): array

Retorna um array com todas as chaves de input em minúsculo ou maiúsculo. Índices numéricos são mantidos como são.

Parâmetros

input

O array a ser modificado

case

Tanto CASE_UPPER quanto CASE_LOWER (padrão)

Valor Retornado

Retorna um array com as chaves em minúsculo ou maiúsculo, ou null se input não for um array.

Exemplos

Exemplo #1 Exemplo da função array_change_key_case()

<?php
$input_array = array("primeiRo" => 1, "segunDo" => 4);
print_r(array_change_key_case($input_array, CASE_UPPER);
?>

O exemplo acima produzirá:

Array
(
    [PRIMEIRO] => 1
    [SEGUNDO] => 2
)

Notas

Nota:

Se uma matriz tem índices que serão os mesmos após a utilização desta função (ex: "chaVE" e "CHave"), o valor que estiver ao final do array irá sobrescrever os outros índices.

Domínio Unix: Unix e UDG

unix:// e udg://

  • unix:///tmp/mysock
  • udg:///tmp/mysock

unix:// provê acessos a conexão via Unix sockets. udg:// provê um transporte alternativo para um socket Unix usando o protocolo de datagramas de usuário (UDP).

Sockets Unix, diferentes de sockets de Internet, não esperam um número de porta. No caso de fsockopen() o parâmetro deve ser o valor 0.

Nota: Sockets Unix domain não são suportados no Windows.

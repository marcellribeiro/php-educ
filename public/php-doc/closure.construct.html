Closure::__construct

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

Closure::__construct — Construtor que desabilita a instanciação

Descrição

private Closure::__construct()

Este método exise somente para desabilitar a instanciação da classe Closure. Objetos desta classe são criados como descrito na página funções anônimas.

Parâmetros

Esta função não possui parâmetros.

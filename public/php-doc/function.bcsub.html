bcsub

(PHP 4, PHP 5, PHP 7, PHP 8)

bcsub — Subtrai um número de precisão arbitrária de outro

Descrição

bcsub(string $num1, string $num2, ?int $scale = null): string

Subtrai num2 de num1.

Parâmetros

num1

O operando da esquerda, como uma string.

num2

O operando da direita, como uma string.

scale

Este parâmetro opcional é usado para definir o número de dígitos após a casa decimal do resultado. Se omitido, o padrão será a escala definida globalmente com a função bcscale() ou retornará para 0 se a escala não tiver sido definida.

Valor Retornado

O resultado da subtração, como uma string.

Changelog

Versão Descrição
8.0.0 scale agora pode ser null.

Exemplos

Exemplo #1 Exemplo de bcsub()

<?php

$a = '1.234';
$b = '5';

echo bcsub($a, $b); // -3
echo bcsub($a, $b, 4); // -3.7660

?>

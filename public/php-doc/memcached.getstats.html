Memcached::getStats

(PECL memcached >= 0.1.0)

Memcached::getStats — Obtem estatísticas do pool de servidores

Descrição

public Memcached::getStats(): array|false

Memcached::getStats() retorna um array contendo o estado de todos os servidores memcache disponíveis. Consulte » especificação do » memcache protocol para obter detalhes sobre essas estatísticas.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Array de estatísticas do servidor, uma entrada por servidor, ou false em caso de falha.

Exemplos

Exemplo #1 Memcached::getStats() exemplo

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

print_r($m->getStats());
?>

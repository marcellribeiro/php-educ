password_get_info

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

password_get_info — Retorna a informações sobre o hash fornecido

Descrição

password_get_info(string $hash): array

Quando for passado um hash válido criado por um algoritmo suportado pela função password_hash(), esta função retornará um array com as informações sobre esse hash.

Parâmetros

hash

Um hash criado por password_hash().

Valor Retornado

Retorna um array associativo com três elementos:

  • algo, que corresponderá a uma constante de algoritmo de password
  • algoName, que terá o nome legível do algoritmo
  • options, que inclui as opções fornecidas quando chamou-se a função password_hash()

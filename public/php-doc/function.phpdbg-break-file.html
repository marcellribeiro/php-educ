phpdbg_break_file

(PHP 5 >= 5.6.3, PHP 7, PHP 8)

phpdbg_break_file — Insere um ponto de interrupção em uma linha de um arquivo

Descrição

phpdbg_break_file(string $file, int $line): void

Insere um ponto de interrupção na line fornecida no file fornecido.

Parâmetros

file

O nome do arquivo.

line

O número da linha.

Valor Retornado

Nenhum valor é retornado.

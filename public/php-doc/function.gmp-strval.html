gmp_strval

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_strval — Converte um número GMP para uma string

Descrição

gmp_strval(resource $gmpnumber, int $base = ?): string

Converte o número GMP para a representação de uma string na base base. A base padrão é 10.

Parâmetros

gmpnumber

O número GMP que será convertido para uma string.

Um objeto GMP, um int ou uma string numérica.

base

A base do número retornado. A base padrão é 10. Valores permitidos para base são de 2 à 36.

Valor Retornado

O número, como uma string.

Exemplos

Exemplo #1 Convertendo um número GMP para uma string

<?php
$a = gmp_init("0x41682179fbf5");
printf("Decimal: %s, 36-based: %s", gmp_strval($a), gmp_strval($a,36));
?>

DirectoryIterator::__toString

(PHP 5, PHP 7, PHP 8)

DirectoryIterator::__toString — Retorna o nome do arquivo como texto

Descrição

public DirectoryIterator::__toString(): string

Retorna o nome do item corrente do DirectoryIterator como texto.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna o nome do item corrente do DirectoryIterator como texto.

Exemplos

Exemplo #1 Exemplo do DirectoryIterator::__toString()

Este exemplo irá listar o conteúdo do diretório o qual o script está rodando.

<?php
$dir = new DirectoryIterator(dirname(__FILE__));
foreach ($dir as $fileinfo) {
echo $fileinfo;
}
?>

O exemplo acima produzirá algo semelhante a:

.
..
maça.jpg
banana.jpg
index.php
pera.jpg

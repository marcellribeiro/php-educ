zip_read

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_read — Lê a próxima entrada em um arquivo ZIPado

Descrição

zip_read(resource $zip): resource

Lê o próximo elemento dentro de uma arquivo ZIP.

Parâmetros

zip

Um arquivo ZIP anteriormente aberto com zip_open().

Valor Retornado

Retorna uma conexao com o arquivo ZIP para ser usado com as funções zip_entry_..., ou false se não houver arquivos para serem lidos, ou um código de erro caso ocorra algum.

ZMQContext::setOpt

(PECL zmq >= 1.0.4)

ZMQContext::setOpt — Define uma opção de socket

Descrição

public ZMQContext::setOpt(integer $key, mixed $value): ZMQContext

Define uma opção de contexto ZMQ. O tipo do parâmetro value depende do parâmetro key. Consulte os Tipos de Constantes ZMQ para mais informações.

Parâmetros

key

Uma das constantes ZMQ::CTXOPT_*.

value

O valor do parâmetro.

Valor Retornado

Retorna o objeto atual. Lança uma exceção do tipo ZMQContextException caso haja erro.

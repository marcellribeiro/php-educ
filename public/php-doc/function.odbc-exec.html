odbc_exec

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_exec — Prepara e executa um comando SQL

Descrição

odbc_exec(resource $connection_id, string $query_string, int $flags = ?): resource

Envia um comando SQL para o servidor de banco de dados.

Parâmetros

connection_id

O identificador da conexão.

query_string

O comando SQL.

flags

Valor Retornado

Retorna um identificador de resultado ODBC se o comando foi executado com sucesso, ou false em erro.

Error::__construct

(PHP 7, PHP 8)

Error::__construct — Constrói o objeto error

Descrição

public Error::__construct(string $message = "", int $code = 0, ?Throwable $previous = null)

Constrói um erro.

Parâmetros

message

A mensagem de erro.

code

O código do erro.

previous

O throwable anterior, utilizado para o encadeamento de exceção.

Notas

Nota:

O parâmetro message NÃO é binary safe.

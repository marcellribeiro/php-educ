Exception::__clone

(PHP 5, PHP 7, PHP 8)

Exception::__clone — Clona a exceção

Descrição

private Exception::__clone(): void

Exceptions não podem ser clonadas, e isso resultará no lançamento de um Error.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Nenhum valor é retornado.

Erros

Exceções não são clonáveis.

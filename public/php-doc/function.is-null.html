is_null

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

is_null — Informa se a variável é null

Descrição

is_null(mixed $var): bool

Verifica se uma dada variável é null.

Parâmetros

var

A variável a ser avaliada.

Valor Retornado

Retorna true se var é null, false caso contrário.

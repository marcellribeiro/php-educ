lcfirst

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

lcfirst — Torna minúsculo o primeiro caractere de uma string

Descrição

lcfirst(string $str): string

Retorna uma string com o primeiro caractere de str em letra minúscula se o caractere for alfabético.

Note que 'alfabético' é determinado pela localidade corrente. Por exemplo, na localidade "C" padrão caracteres tais como a-umlaut (ä) não serão convertidos.

Parâmetros

str

A string de entrada.

Valor Retornado

Retorna a string resultante.

Exemplos

Exemplo #1 Exemplo de lcfirst()

<?php
$foo = 'OlaMundo';
$foo = lcfirst($foo); // olaMundo

$bar = 'OLA MUNDO!';
$bar = lcfirst($bar); // oLLA MUNDO!
$bar = lcfirst(strtoupper($bar)); // oLLA MUNDO!
?>

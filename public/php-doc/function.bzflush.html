bzflush

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

bzflush — Sem efeitos

Descrição

bzflush(resource $bz): bool

Força a escrita de todos os dados do bzip2 que estão em buffer para o ponteiro de arquivo bz, mas é implementado como uma função vazia na libbz2, e portanto não gera efeitos.

Parâmetros

bz

O ponteiro do arquivo. Ele precisa ser válido e precisa apontar para um arquivo aberto com sucesso por bzopen().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

libxml_set_streams_context

(PHP 5, PHP 7, PHP 8)

libxml_set_streams_context — Define os contextos streams para o próximo documento carregado ou escrito na libxml

Descrição

libxml_set_streams_context(resource $streams_context): void

Define contextos streams para o próximo documento carregado ou escrito na libxml.

Parâmetros

streams_context

O resource do contexto stream (criado com stream_context_create())

Valor Retornado

Nenhum valor é retornado.

Exemplos

Exemplo #1 Um exemplo da libxml_set_streams_context()

<?php

$opts = array(
'http' => array(
'user_agent' => 'PHP libxml agent',
)
);

$context = stream_context_create($opts);
libxml_set_streams_context($context);

// request a file through HTTP
$doc = DOMDocument::load('http://www.example.com/file.xml');

?>

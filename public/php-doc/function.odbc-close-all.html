odbc_close_all

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_close_all — Fecha todas as conexões ODBC

Descrição

odbc_close_all(): void

odbc_close_all() irá fechar todas as conexões para os servidores de banco de dados.

Valor Retornado

Nenhum valor é retornado.

Notas

Nota:

Esta função irá falhar se houverem transações abertas na conexão. Neste caso a conexão permanecerá aberta.

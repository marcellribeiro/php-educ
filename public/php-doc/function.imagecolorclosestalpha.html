imagecolorclosestalpha

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

imagecolorclosestalpha — Retorna o índice da cor mais próxima da cor especificada + transparencia

Descrição

imagecolorclosestalpha(
    resource $image,
    int $red,
    int $green,
    int $blue,
    int $alpha
): int

Retorna o índice da cor da paleta da imagem o qual seja mais "próximo" do valor RGB e de alpha level.

Veja também imagecolorexactalpha().

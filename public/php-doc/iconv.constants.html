Constantes pré-definidas

É possível identificar em tempo de execução qual implementação do iconv é adotada por esta extensão.

Constantes de implementação iconv
Nome Tipo Descrição
ICONV_IMPL string O nome da implementação
ICONV_VERSION string A versão de implementação

Nota:

Escrever scripts dependentes de implementação com essas constantes é fortemente desencorajado.

As seguintes constantes são também disponível:

Other iconv constants
Nome Tipo Descrição
ICONV_MIME_DECODE_STRICT int Uma máscara de bits usada para iconv_mime_decode()
ICONV_MIME_DECODE_CONTINUE_ON_ERROR int Uma máscara de bits usada para iconv_mime_decode()

Introdução

A extensão mysqli permite a você acessar a funcionalidade provida pelo MySQL 4.1 e posterior. Maiores informações sobre o servidor de banco de dados MySQL podem ser encontradas em » http://www.mysql.com/

Uma visão geral de software disponível para usar MySQL através do PHP pode ser encontrada em Overview

Documentação para o MySQL pode ser encontrada em » http://dev.mysql.com/doc/.

Partes desta documentação foram incluídas do manual do MySQL com a permissão de Oracle Corporation.

Os exemplos usam tanto o banco de dados » world quanto o banco de dados » sakila, que estão disponíveis livremente.

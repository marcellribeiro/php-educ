socket_wsaprotocol_info_import

(PHP 7 >= 7.3.0, PHP 8)

socket_wsaprotocol_info_import — Imports a Socket from another Process

Descrição

socket_wsaprotocol_info_import(string $info_id): Socket|false

Imports a socket which has formerly been exported from another process.

Nota: This function is available only on Windows.

Parâmetros

info_id

The ID which has been returned by a former call to socket_wsaprotocol_info_export().

Valor Retornado

Returns a Socket instance on success, ou false em caso de falha

Changelog

Versão Descrição
8.0.0 On success, this function returns a Socket instance now; previously, a resource was returned.

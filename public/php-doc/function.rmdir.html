rmdir

(PHP 4, PHP 5, PHP 7, PHP 8)

rmdir — Remove um diretório

Descrição

rmdir(string $dirname, resource $context = ?): bool

Tenta remover o diretório com o nome de dirname. O diretório tem que estar vazio e as permissões relevantes autorizarem a esta operação.

Parâmetros

dirname

Caminho para o diretório.

context

Um resource de contexto de stream.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Changelog

Versão Descrição
5.0.0 No PHP 5.0.0 rmdir() pode também ser usada com alguns URL wrappers. Consulte Protocolos e Wrappers suportados para uma listagem de wrappers suportados pela rmdir().

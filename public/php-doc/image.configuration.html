Configurações em Execução

O comportamento dessas funções é afetado pelas configurações do php.ini.

Image Opções de Configuração
Nome Padrão Modificável Changelog
gd.jpeg_ignore_warning "1" PHP_INI_ALL Disponível a partir do PHP 5.1.3.
Para mais detalhes e definições dos modos PHP_INI_*, veja os Onde uma configuração deve ser definida.

Aqui está uma breve explicação das diretivas de configuração.

gd.jpeg_ignore_warning bool

Ignora os avisos (mas não os erros) criado por libjpeg(-turbo).

Changelog para gd.jpeg_ignore_warning
Versão Descrição
7.1.0 O padrão de gd.jpeg_ignore_warning foi alterado de 0 para 1.

Veja também as diretivas de configuração exif.

Aviso

As funções para imagem usam bastante memória. Certifique-se de definir memory_limit grande o bastante, se você estiver usando a versão em pacote da biblioteca GD.

fmod

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

fmod — Retorna o resto em ponto flutuante (módulo) da divisão dos argumentos

Descrição

fmod(float $x, float $y): float

Retorna o resto em ponto flutuante da divisão do dividendo (x) pelo divisor (y). O resto (r) é definido como: x = i * y + r, para algum inteiro i. Se o y é diferente de zero, r terá o mesmo sinal de x e também uma magnitude menor do que a magnitude de y.

Parâmetros

x

O dividendo

y

O divisor

Valor Retornado

O resto em ponto flutuante de x/y

Exemplos

Exemplo #1 Usando fmod()

<?php
$x = 5.7;
$y = 1.3;
$r = fmod($x, $y);
// $r equals 0.5, because 4 * 1.3 + 0.5 = 5.7
?>

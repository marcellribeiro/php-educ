iconv_strlen

(PHP 5, PHP 7, PHP 8)

iconv_strlen — Retorna a contagem de caracteres da string

Descrição

iconv_strlen(string $string, ?string $encoding = null): int|false

Ao contrário de strlen(), iconv_strlen() conta as ocorrências de caracteres na string de sequência de bytes fornecida com base no conjunto de caracteres especificado, cujo resultado não é necessariamente idêntico ao comprimento da string em bytes.

Parâmetros

string

A string.

encoding

Se o parâmetro encoding for omitido ou null, presume-se que a string esteja codificada em iconv.internal_encoding.

Valor Retornado

Retorna a contagem de caracteres da string, como um número inteiro, ou false se ocorrer um erro durante a codificação.

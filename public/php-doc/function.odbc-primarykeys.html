odbc_primarykeys

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_primarykeys — Obtém a chave primária de uma tabela

Descrição

odbc_primarykeys(
    resource $connection_id,
    string $qualifier,
    string $owner,
    string $table
): resource

Retorna um identificador de resultado que pode ser usado para obter o nome das colunas chave primária de uma tabela.

Parâmetros

connection_id

O identificador da conexão.

qualifier

owner

table

Valor Retornado

Retorna um identificador de resultado ODBC ou false em falha.

O conjunto de resultados tem as seguintes colunas:

  • TABLE_QUALIFIER
  • TABLE_OWNER
  • TABLE_NAME
  • COLUMN_NAME
  • KEY_SEQ
  • PK_NAME

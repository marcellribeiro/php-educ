mysqli::set_charset

mysqli_set_charset

(PHP 5 >= 5.0.5, PHP 7, PHP 8)

mysqli::set_charset -- mysqli_set_charset — Define o conjunto de caracteres do cliente

Descrição

Estilo orientado a objetos

public mysqli::set_charset(string $charset): bool

Estilo procedural

mysqli_set_charset(mysqli $mysql, string $charset): bool

Define o conjunto de caracteres a ser usado ao enviar dados de e para o servidor de banco de dados.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

charset

O conjunto de caracteres desejado.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Erros

Se o relatório de erros da extensão mysqli estiver habilitado (MYSQLI_REPORT_ERROR) e a operação solicitada falhar, um aviso será gerado. Se, além disso, o modo for definido como MYSQLI_REPORT_STRICT, uma exceção mysqli_sql_exception será lançada.

Exemplos

Exemplo #1 mysqli::set_charset() exemplo

Estilo orientado a objetos

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$mysqli = new mysqli("localhost", "my_user", "my_password", "test");

printf("Conjunto de caracteres inicial: %s\n", $mysqli->character_set_name());

/* alterar conjunto de caracteres para utf8mb4 */
$mysqli->set_charset("utf8mb4");

printf("Conjunto de caracteres atual: %s\n", $mysqli->character_set_name());

Estilo procedural

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$link = mysqli_connect('localhost', 'my_user', 'my_password', 'test');

printf("Conjunto de caracteres inicial: %s\n", mysqli_character_set_name($link));

/* alterar conjunto de caracteres para utf8mb4 */
mysqli_set_charset($link, "utf8mb4");

printf("Conjunto de caracteres atual: %s\n", mysqli_character_set_name($link));

Os exemplos acima produzirão algo semelhante a:

Conjunto de caracteres inicial: latin1
Conjunto de caracteres atual: utf8mb4

Notas

Nota:

Para usar esta função em uma plataforma Windows, você precisa da biblioteca cliente MySQL versão 4.1.11 ou superior (para MySQL 5.0, você precisa da versão 5.0.6 ou superior).

Nota:

Esta é a maneira preferida de alterar o conjunto de caracteres. Usar mysqli_query() para configurá-lo (como SET NAMES utf8) não é recomendado. Consulte a seção de conceitos do conjunto de caracteres do MySQL para obter mais informações.

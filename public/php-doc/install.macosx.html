Instalação no macOS

Índice

  • Usando Pacotes
  • Usando o pacote do PHP no macOS para versões anteriores ao macOS Monterey
  • Compilando o PHP no MacOS

Essa seção contém notas e dicas específicas para instalação do PHP no macOS X. O PHP já vem incluído nos macOS entre o macOS X (10.0.0) e o macOS Monterey (12.0.0). A compilação do PHP no macOS é semelhante à instalação em Unix.

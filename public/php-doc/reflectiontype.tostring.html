ReflectionType::__toString

(PHP 7, PHP 8)

ReflectionType::__toString — To string

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 7.1.0. O uso desta função é altamente desaconselhado.

Descrição

public ReflectionType::__toString(): string

Gets the parameter type name.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Returns the type of the parameter.

Changelog

Versão Descrição
7.1.0 ReflectionType::__toString() has been deprecated.

Exemplos

Exemplo #1 ReflectionType::__toString() example

<?php
function someFunction(string $param) {}

$reflectionFunc = new ReflectionFunction('someFunction');
$reflectionParam = $reflectionFunc->getParameters()[0];

echo $reflectionParam->getType();

O exemplo acima produzirá algo semelhante a:

string

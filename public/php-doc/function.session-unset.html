session_unset

(PHP 4, PHP 5, PHP 7, PHP 8)

session_unset — Libera todas as variáveis de sessão

Descrição

session_unset(): void

A função session_unset() libera todas as variáveis atualmente registradas na sessão.

Valor Retornado

Nenhum valor é retornado.

Notas

Nota:

Se $_SESSION (ou $HTTP_SESSION_VARS para PHP 4.0.6 ou inferior) for usada, use unset() para desregistrar a variável da sessão, isto é, unset ($_SESSION['varname']);.

Cuidado

NÃO elimine completamente $_SESSION com unset($_SESSION) já que isto desabilitará o registro de variáveis de sessão através da super global $_SESSION.

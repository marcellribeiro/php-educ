dbase_delete_record

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_delete_record — Exclui um registro do banco de dados

Descrição

dbase_delete_record(int $dbase_identifier, int $record_number): bool

Marca o registro indicado para ser excluído do banco de dados.

Nota:

Para realmente remover o registro do banco de dados, você deve também utilizar dbase_pack().

Parâmetros

dbase_identifier

Identificador do banco de dados, retornado por dbase_open() ou dbase_create().

record_number

Um inteiro que vai de 1 até o número de registros no banco de dados (como retornado por dbase_numrecords()).

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

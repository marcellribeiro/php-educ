mysqli::release_savepoint

mysqli_release_savepoint

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

mysqli::release_savepoint -- mysqli_release_savepoint — Remove o ponto de salvamento nomeado do conjunto de pontos de salvamento da transação atual

Descrição

Estilo orientado a objetos

public mysqli::release_savepoint(string $name): bool

Estilo procedural:

mysqli_release_savepoint(mysqli $mysql, string $name): bool

Esta função é idêntica à execução de $mysqli->query("RELEASE SAVEPOINT `$name`");. Esta função não aciona commit ou rollback.

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

name

O identificador do ponto de salvamento.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

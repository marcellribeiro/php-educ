dbase_get_record_with_names

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_get_record_with_names — Obtém um registro do banco de dados como uma matriz associativa.

Descrição

dbase_get_record_with_names(int $dbase_identifier, int $record_number): array

Obtém um registro do banco de dados como uma matriz associativa.

Parâmetros

dbase_identifier

Identificador do banco de dados, retornado por dbase_open() ou dbase_create().

record_number

O índice do registro.

Valor Retornado

Uma matriz associativa com o registro. Isto também irá incluir uma chave chamada deleted a qual é definida como 1 se o registro foi marcado para exclusão (veja dbase_delete_record()).

Cada campo é convertido para o tipo PHP apropriado, exceto:

  • Datas são mantidas como strings.
  • Inteiros que causariam uma sobrecarga (> 32 bits) são retornados como uma string.

Em caso de erro, dbase_get_record_with_names() irá retornar false.

Exemplos

Exemplo #1 Listando todos os membros registrados no banco de dados

<?php
// open in read-only mode
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
$record_numbers = dbase_numrecords($db);
for ($i = 1; $i <= $record_numbers; $i++) {
$row = dbase_get_record_with_names($db, $i);
if ($row['ismember'] == 1) {
echo "Member #$i: " . trim($row['name']) . "\n";
}
}
}
?>

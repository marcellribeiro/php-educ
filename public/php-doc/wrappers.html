Protocolos e Wrappers suportados

O PHP vem com muitos wrappers para vários protocolos URL-style para uso com as funções do sistema de arquivos, tais como fopen(), copy(), file_exists() e filesize(). Além desses wrappers é possível registrar wrappers customizados usando a função stream_wrapper_register().

Nota: A sintaxe de URL usada para descrever um wrapper suporta apenas a sintaxe scheme://.... As sintaxes scheme:/ e scheme: não são suportadas.

Índice

  • file:// — Accessing local filesystem
  • http:// — Accessing HTTP(s) URLs
  • ftp:// — Accessing FTP(s) URLs
  • php:// — Accessing various I/O streams
  • zlib:// — Compression Streams
  • data:// — Data (RFC 2397)
  • glob:// — Find pathnames matching pattern
  • phar:// — PHP Archive
  • ssh2:// — Secure Shell 2
  • rar:// — RAR
  • ogg:// — Audio streams
  • expect:// — Process Interaction Streams

svn_client_version

(PECL svn >= 0.1.0)

svn_client_version — Returns the version of the SVN client libraries

Descrição

svn_client_version(): string

Returns the version of the SVN client libraries

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

String version number, usually in form of x.y.z.

Exemplos

Exemplo #1 Basic example

<?php
echo svn_client_version();
?>

O exemplo acima produzirá algo semelhante a:

1.3.1

Notas

Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome e documentação poderão mudar sem aviso prévio em futuras versões do PHP. Use por sua conta e risco.

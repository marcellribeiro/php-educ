get_magic_quotes_gpc

(PHP 4, PHP 5, PHP 7)

get_magic_quotes_gpc — Obtém a configuração atual de magic_quotes_gpc

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 7.4.0 e foi REMOVIDA a partir do PHP 8.0.0. O uso desta função é altamente desaconselhado.

Descrição

get_magic_quotes_gpc(): bool

Sempre retorna false.

Valor Retornado

Sempre retorna false.

Changelog

Versão Descrição
7.4.0 Esta função está obsoleta.

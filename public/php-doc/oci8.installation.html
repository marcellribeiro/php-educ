Instalação

Você tem que compilar o PHP com a opção --with-oci8[=DIR], onde DIR é a variável de ambiente ORACLE_HOME.

Se você está usando » Oracle Instant Client, você precisa compilar o PHP com a opção --with-oci8=instantclient,/path/to/oic/lib. Note que suporte a Oracle Instant Client apareceu primeiro nas versões 4.3.11 e 5.0.4.

Nota:

Note que nas versões da OCI8 antes da 1.1, você tem que usar a opção --with-oci8-instant-client para configurar o PHP com Oracle Instant Client.

str_contains

(PHP 8)

str_contains — Determina se uma string contém uma substring fornecida

Descrição

str_contains(string $haystack, string $needle): bool

Realiza uma busca case-sensitive indicando se needle existe em haystack.

Parâmetros

haystack

A string onde será feita a busca.

needle

A substring para ser buscada em haystack.

Valor Retornado

Retorna true se needle está em haystack, false caso contrário.

Exemplos

Exemplo #1 Usando a string vazia ''

<?php
if (str_contains('abc', '')) {
echo "Verificando a existência de uma string vazia sempre retornará true";
}
?>

O exemplo acima produzirá:

Verificando a existência de uma string vazia sempre retornará true

Exemplo #2 Mostrando o case-sensitivity

<?php
$string = 'A raposa preguiçosa pulou a cerca';

if (str_contains($string, 'raposa')) {
echo "A string 'raposa' foi encontrada na string\n";
}

if (str_contains($string, 'Raposa')) {
echo 'A string "Raposa" foi encontrada na string';
} else {
echo '"Raposa" não foi encontrada porque o case não corresponde';
}

?>

O exemplo acima produzirá:

A string 'raposa' foi encontrada na string
"Raposa" não foi encontrada porque o case não corresponde

Notas

Nota: Esta função é binary-safe.

apache_response_headers

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

apache_response_headers — Obtém todos os cabeçalhos da resposta HTTP

Descrição

apache_response_headers(): array|false

Obtém todos os cabeçalhos de resposta. Funciona na CLI e nos servidores web Apache, FastCGI e FPM.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Um array de todos os cabeçalhos de resposta do Apache em caso de sucesso, ou ou false em caso de falha.

Exemplos

Exemplo #1 Exemplo da função apache_response_headers()

<?php
print_r(apache_response_headers());
?>

O exemplo acima produzirá algo semelhante a:

Array
(
    [Accept-Ranges] => bytes
    [X-Powered-By] => PHP/4.3.8
)

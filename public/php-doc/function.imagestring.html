imagestring

(PHP 4, PHP 5, PHP 7, PHP 8)

imagestring — Desenha uma string horizontalmente

Descrição

imagestring(
    resource $image,
    int $font,
    int $x,
    int $y,
    string $string,
    int $color
): bool

Desenha uma string nas dadas coordenadas.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

font

Pode ser 1, 2, 3, 4, 5 para as fontes nativas na codificação latin2 (onde números maiores correspondem a fontes maiores) ou uma instância de GdFont, retornada por imageloadfont().

x

Coordenada x superior à esquerda

y

Coordenada y superior à esquerda

string

A string a ser escrita

color

Um identificador de cor criado com imagecolorallocate()

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Exemplo da imagestring()

<?php
// cria uma imagem de 100*30
$im = imagecreate(100, 30);

// fundo branco e texto azul
$bg = imagecolorallocate($im, 255, 255, 255);
$textcolor = imagecolorallocate($im, 0, 0, 255);

// escreve a string em cima na esquerda
imagestring($im, 5, 0, 0, "Hello world!", $textcolor);

// envia a imagem
header("Content-type: image/png");
imagepng($im);
?>

ibase_blob_close

(PHP 5, PHP 7 < 7.4.0)

ibase_blob_close — Fecha o blob

Descrição

ibase_blob_close(resource $blob_handle): mixed

Esta função fecha um BLOB que tenha sido aberto para leitura com ibase_open_blob() ou aberto para escrita com ibase_create_blob().

Parâmetros

blob_handle

Um manipulador BLOB aberto com ibase_create_blob() ou ibase_open_blob().

Valor Retornado

Se o BLOB foi lido, esta função retorna true em sucesso, se o BLOB foi escrito, esta função retorna uma string contendo o id do BLOB que foi definido para ele pelo banco de dados. Em falha, esta função retorna false.

Instalação

Não há necessidade de instalação para usar essas funções; elas fazem parte do núcleo do PHP.

Nota: Recuperando o banco de dados atualizado de fusos horários

A versão mais recente do banco de dados de fusos horários pode ser instalada com o pacote PECL » timezonedb.

Nota: Suporte experimental ao DateTime no PHP 5.1.x

Apesar da classe DateTime (e funções relacionadas) estarem disponíveis por padrão desde o PHP 5.2.0, é possível adicionar o suporte experimental no PHP 5.1.x utilizando a seguinte flag antes de configurar/compilar: CFLAGS=-DEXPERIMENTAL_DATE_SUPPORT=1

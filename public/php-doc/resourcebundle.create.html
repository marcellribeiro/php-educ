ResourceBundle::create

resourcebundle_create

ResourceBundle::__construct

(PHP 5 >= 5.3.2, PHP 7, PHP 8, PECL intl >= 2.0.0)

ResourceBundle::create -- resourcebundle_create -- ResourceBundle::__construct — Cria um pacote de recursos

Descrição

Estilo orientado a objetos (method)

public static ResourceBundle::create(string $locale, string $bundlename, bool $fallback = ?): ResourceBundle

Estilo procedural

resourcebundle_create(string $locale, string $bundlename, bool $fallback = ?): ResourceBundle

Estilo orientado a objetos (constructor):

public ResourceBundle::__construct(string $locale, string $bundlename, bool $fallback = ?)

Cria um pacote de recursos.

Parâmetros

locale

Localidade para a qual os recursos devem ser carregados (nome da localidade, ex. en_CA).

bundlename

O diretório onde os dados estão armazenados ou o nome do arquivo .dat.

fallback

Indica se a localidade deve ser uma correspondência exata ou se a retroação para a localidade base é permitida.

Valor Retornado

Retorna um objeto ResourceBundle ou null caso haja erro.

Exemplos

Exemplo #1 Exemplo de resourcebundle_create()

<?php
$r = resourcebundle_create( 'es', "/usr/share/data/meuapp");
echo $r['stringteste'];
?>

Exemplo #2 Exemplo de ResourceBundle::create()

<?php
$r = ResourceBundle::create( 'es', "/usr/share/data/meuapp");
echo $r['stringteste'];
?>

O exemplo acima produzirá:

¡Hola, mundo!

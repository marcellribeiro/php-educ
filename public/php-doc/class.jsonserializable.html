A interface JsonSerializable

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

Introdução

Os objetos que implementam JsonSerializable podem personalizar sua representação JSON quando codificados com json_encode().

Resumo da Interface

interface JsonSerializable {
/* Métodos */
public jsonSerialize(): mixed
}

Índice

  • JsonSerializable::jsonSerialize — Especifique os dados que devem ser serializados para JSON

curl_multi_strerror

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_multi_strerror — Retorna uma string que descreve o código de erro

Descrição

curl_multi_strerror(int $error_code): ?string

Retorna um texto descrevendo o código de erro de CURLM informado como parâmetro.

Parâmetros

error_code

Uma das constantes de » códigos de erros de CURLM.

Valor Retornado

Retorna a descrição do erro para um código de erro válido, caso contrário null.

Exemplos

Exemplo #1 Exemplo de curl_multi_strerror()

<?php
// Cria os manipuladores cURL
$ch1 = curl_init("http://example.com/");
$ch2 = curl_init("http://php.net/");

// Cria um "multi" manipulador cURL
$mh = curl_multi_init();

// Adiciona os manipuladores ao "multi" manipulador
curl_multi_add_handle($mh, $ch1);
curl_multi_add_handle($mh, $ch2);

// Executa o "multi" manipulador
do {
$status = curl_multi_exec($mh, $active);
if ($active) {
curl_multi_select($mh);
}
} while ($active && $status === CURLM_OK);

// Verifica se há erros
if ($status != CURLM_OK) {
// Exibe mensagem de erro
echo "ERROR!\n " . curl_multi_strerror($status);
}

?>

socket_close

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

socket_close — Fecha um socket resource

Descrição

socket_close(resource $socket): void
Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome e documentação poderão mudar sem aviso prévio em futuras versões do PHP. Use por sua conta e risco.

socket_close() fecha o socket resource dado por socket.

Nota:

socket_close() não pode ser usado em arquivos PHP file resources criados com fopen(), popen(), fsockopen(), ou pfsockopen(); isso serve parapara sockets criados com socket_create() ou socket_accept().

Veja também socket_bind(), socket_listen(), socket_create() e socket_strerror().

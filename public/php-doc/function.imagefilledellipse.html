imagefilledellipse

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

imagefilledellipse — Desenha um elipse prenchido

Descrição

imagefilledellipse(
    resource $image,
    int $cx,
    int $cy,
    int $w,
    int $h,
    int $color
): bool

imagefilledellipse() desenha um elipse centrado em cx, cy (em cima na esquerda é 0, 0) na imagem representada por image. W e h especifica a largura e altura do elipse respectivamente. O elipse é prenchido com a cor color. Retorna true em caso de sucesso ou false em caso de falha.

Nota:

Esta função foi adicionada no PHP 4.0.6 e precisa da GD 2.0.1 ou posterior.

Exemplo #1 Exemplo imagefilledellipse()

<?php

// cria uma imagem em branco
$image = imagecreate(400, 300);

// prenche a cor do fundo
$bg = imagecolorallocate($image, 0, 0, 0);

// escolhe uma cor para o elipse
$col_ellipse = imagecolorallocate($image, 255, 255, 255);

// desenha o elipse branco
imagefilledellipse($image, 200, 150, 300, 200, $col_ellipse);

// mostra a imagem
header("Content-type: image/png");
imagepng($image);

?>

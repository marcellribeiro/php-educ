CURLStringFile::__construct

(PHP 8 >= 8.1.0)

CURLStringFile::__construct — Crie um objeto CURLStringFile

Descrição

public CURLStringFile::__construct(string $data, string $postname, string $mime = "application/octet-stream")

Cria um objeto CURLStringFile, usado para carregar um arquivo com CURLOPT_POSTFIELDS.

Parâmetros

data

O conteúdo a ser carregado.

postname

O nome do arquivo a ser usado nos dados de upload.

mime

Tipo MIME do arquivo (o padrão é application/octet-stream).

Exemplos

Exemplo #1 CURLStringFile::__construct() exemplo

<?php
/* http://example.com/upload.php:
<?php
var_dump($_FILES);
var_dump(file_get_contents($_FILES['test_string']['tmp_name']));
?>
*/

// Criar um identificador cURL
$ch = curl_init('http://example.com/upload.php');

// Criar um objeto CURLStringFile
$cstringfile = new CURLStringFile('conteúdo de upload de teste','test.txt','text/plain');

// Atribuir dados POST
$data = array('test_string' => $cstringfile);
curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, $data);

// Execute o identificador
curl_exec($ch);
?>

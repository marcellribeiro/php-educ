Suporte a sessões

Memcached fornece um manipulador de sessão personalizado que pode ser usado para armazenar sessões de usuário no memcache. Uma instância memcached completamente separada é usada para isso internamente, então você pode usar um pool de servidores diferente, se necessário. As keys de sessão são armazenadas sob o prefixo memc.sess.key., portanto, esteja ciente disso se usar o mesmo conjunto de servidores para sessões e armazenamento em cache genérico.

session.save_handler string

Defina como memcached para ativar o suporte a sessões.

session.save_path string

Defina uma vírgula separada das entradas hostname:port a serem usadas para o pool de servidores de sessão, por exemplo, "sess1:11211, sess2:11211".

posix_ttyname

(PHP 4, PHP 5, PHP 7, PHP 8)

posix_ttyname — Determine terminal device name

Descrição

posix_ttyname(resource|int $file_descriptor): string|false

Returns a string for the absolute path to the current terminal device that is open on the file descriptor file_descriptor.

Parâmetros

fd

O descritor de arquivo, que deve ser um resource de arquivo ou um int. Um int será interpretado como um descritor de arquivo que pode ser passado diretamente à chamada de sistema subjacente.

Na maioria dos casos, um resource de arquivo deve ser fornecido.

Valor Retornado

On success, returns a string of the absolute path of the file_descriptor. On failure, returns false

Introdução

O PHP 5 vem com uma API completa de reflexão que acrescenta a capacidade de realizar engenharia reversa em classes, interfaces, funções, métodos e extensões. Além disso, a API de reflexão oferece maneiras de recuperar comentários de documentação para as funções, classes e métodos.

Por favor note que algumas partes da API está faltando o código necessário para executar a extensão de reflexão. Por exemplo, uma classe PHP interna pode estar faltando dados de reflexão para as propriedades. Estes são poucos casos que são considerados bugs, não entando, devem ser descobertos e corrigidos.

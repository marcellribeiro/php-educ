gmp_mod

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_mod — Operação Módulo

Descrição

gmp_mod(resource $n, resource $d): resource

Calcula n módulo d. O resultado é sempre positivo, o sinal de d é ignorado.

Parâmetros

n

Um objeto GMP, um int ou uma string numérica.

d

O módulo que será calculado.

Um objeto GMP, um int ou uma string numérica.

Valor Retornado

Um objeto GMP.

Exemplos

Exemplo #1 Exemplo da gmp_mod()

<?php
$mod = gmp_mod("8", "3");
echo gmp_strval($mod) . "\n";
?>

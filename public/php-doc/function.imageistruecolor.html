imageistruecolor

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

imageistruecolor — Indica quando uma imagem é uma imagem truecolor

Descrição

imageistruecolor(resource $image): bool

imageistruecolor() indica quando a imagem image é uma imagem truecolor.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

Valor Retornado

Retorna true se o image é truecolor, false caso contrário.

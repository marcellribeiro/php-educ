Suporte do Windows

configure flags

configure agora considera CFLAGS e LDFLAGS adicionais definidos como variáveis de ambiente.

CTRL handling

CTRL+C e CTRL+BREAK no console podem ser capturados definindo uma função de manipulador com sapi_windows_set_ctrl_handler().

proc_open() no Windows pode receber uma opção "create_process_group". É necessário se o processo filho deve lidar com eventos CTRL.

OPcache

O OPcache agora suporta um número arbitrário de caches separados por usuário por meio da diretiva INI opcache.cache_id. Todos os processos com o mesmo ID de cache e usuário compartilham uma instância do OPcache.

stat

A implementação stat foi refatorada.

  • Um número de inode é entregue e é baseado no índice de arquivo NTFS.
  • O número do dispositivo agora é baseado no número de série do volume.

Observe que ambos os valores são derivados do sistema e fornecidos como estão em sistemas de 64 bits. Em sistemas de 32 bits, esses valores podem ultrapassar o número inteiro de 32 bits no PHP, portanto, são falsos.

libsqlite3

libsqlite3 não é mais compilado estaticamente em php_sqlite3.dll e php_pdo_sqlite.dll, mas sim disponível como libsqlite3.dll. Consulte as instruções de instalação para SQLite3 e PDO_SQLITE, respectivamente.

checkdnsrr

(PHP 4, PHP 5, PHP 7, PHP 8)

checkdnsrr — Checa os registros DNS correspondentes para o nome do host ou endereço IP informado

Descrição

checkdnsrr(string $host, string $type = "MX"): bool

Pesquisa o DNS para os registros type correspondentes host.

Parâmetros

host

host pode ser o endereço IP em notação quádrupla separada por pontos ou o nome do host.

type

type pode ser: A, MX, NS, SOA, PTR, CNAME, AAAA, A6, SRV, NAPTR, TXT or ANY.

Valor Retornado

Retorna true se o registro for encontrado; retorna false se não encontrar registros ou um erro ocorrer.

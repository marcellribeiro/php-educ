dbase_get_record

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_get_record — Obtém um registro do banco de dados como uma matriz indexada

Descrição

dbase_get_record(int $dbase_identifier, int $record_number): array

Obtém um registro do banco de dados como uma matriz indexada.

Parâmetros

dbase_identifier

O identificador do banco de dados, retornado por dbase_open() ou dbase_create().

record_number

O índice do registro.

Valor Retornado

Uma matriz indexada com o registro. Esta matriz também irá incluir uma chave associativa chamada deleted a qual é definida como 1 se o registro foi marcado para exclusão (veja dbase_delete_record()).

Cada campo é convertido para o tipo PHP apropriado, exceto:

  • Datas são mantidas como strings.
  • Inteiros que causariam uma sobrecarga (> 32 bits) são retornados como uma string.

Em caso de erro, dbase_get_record() irá retornar false.

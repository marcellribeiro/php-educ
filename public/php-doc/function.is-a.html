is_a

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

is_a — Verifica se um objeto é ou herda de uma classe

Descrição

is_a(mixed $object_or_class, string $class, bool $allow_string = false): bool

Verifica se o object_or_class é de um tipo de objeto ou de um subtipo.

Parâmetros

object_or_class

O nome da classe ou o objeto.

class

O nome da classe.

allow_string

Se esse parâmetro é configurado para false, um nome de classe em string em object_or_class não será permitido. Isso evita de chamar o autoloader de uma classe inexistente.

Valor Retornado

Retorna true se o objeto é uma instância do tipo ou se o tipo é um tipo implementado em herança, false nos demais casos.

Exemplos

Exemplo #1 Exemplo de is_a()

<?php
// Define uma classe
class WidgetFactory
{
var $oink = 'moo';
}

// Cria um objeto
$WF = new WidgetFactory();

if (is_a($WF, 'WidgetFactory')) {
echo "\$WF implementa WidgetFactory\n";
}
?>

Exemplo #2 Utilizando o operador instanceof

<?php
if ($WF instanceof WidgetFactory) {
echo "\$WF implementa WidgetFactory\n";
}
?>

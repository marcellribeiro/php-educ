Memcached::getServerList

(PECL memcached >= 0.1.0)

Memcached::getServerList — Obtenha a lista dos servidores no pool

Descrição

public Memcached::getServerList(): array

Memcached::getServerList() retorna a lista de todos os servidores que estão em seu pool de servidores.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

A lista de todos os servidores no pool de servidores.

Exemplos

Exemplo #1 Memcached::getServerList() exemplo

<?php
$m = new Memcached();
$m->addServers(array(
array('mem1.domain.com', 11211, 20),
array('mem2.domain.com', 11311, 80),
));
var_dump($m->getServerList());
?>

O exemplo acima produzirá:

array(2) {
  [0]=>
  array(3) {

    string(15) "mem1.domain.com"

    int(11211)

    int(20)

  [1]=>
  array(3) {
    ["host"]=>
    string(15) "mem2.domain.com"
    ["port"]=>
    int(11311)
    ["weight"]=>
    int(80)
  }
}

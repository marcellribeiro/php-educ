imagefttext

(PHP 4 >= 4.0.7, PHP 5, PHP 7, PHP 8)

imagefttext — Escreve texto na imagem usando fontes FreeType 2

Descrição

imagefttext(
    resource $image,
    float $size,
    float $angle,
    int $x,
    int $y,
    int $col,
    string $font_file,
    string $text,
    array $extrainfo = ?
): array
Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

size

angle

x

y

col

font_file

text

extrainfo

Valor Retornado

Esta função retorna um array contendo os seguintes elementos:

0 coordenada x inferior à esquerda
1 coordenada y inferior à esquerda
2 coordenada x inferior à direita
3 coordenada y inferior à direita
4 coordenada x superior à direita
5 coordenada y superior à direita
6 coordenada x superior à esquerda
7 coordenada y superior à esquerda

Notas

Nota: Esta função só estará disponível se o PHP for compilado com suporte ao freetype (--with-freetype-dir=DIR).

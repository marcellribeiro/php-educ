ibase_drop_db

(PHP 5, PHP 7 < 7.4.0)

ibase_drop_db — Exclui um banco de dados

Descrição

ibase_drop_db(resource $connection = ?): bool

Esta função exclui um banco de dados que foi aberto por ibase_connect() ou ibase_pconnect(). O banco de dados é fechado e excluído do servidor.

Parâmetros

connection

Um link identificador do InterBase. Se omitido, o último link aberto é assumido.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

json_last_error_msg

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

json_last_error_msg — Retorna uma string contento a mensagem de erro da última chamada de json_encode() ou json_decode()

Descrição

json_last_error_msg(): string

Retorna uma string contento a mensagem de erro da última chamada de json_encode() ou json_decode() , que não especificou JSON_THROW_ON_ERROR.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna a mensagem de erro em caso de sucesso, ou "No error" se não ocorreu erro.

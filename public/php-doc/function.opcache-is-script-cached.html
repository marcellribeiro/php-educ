opcache_is_script_cached

(PHP 5 >= 5.5.11, PHP 7, PHP 8, PECL ZendOpcache >= 7.0.4)

opcache_is_script_cached — Informa se um script é armazenado em cache no OPCache

Descrição

opcache_is_script_cached(string $filename): bool

Esta função verifica se um script PHP foi armazenado em cache no OPCache. Isso pode ser usado para detectar mais facilmente o "aquecimento" do cache para um determinado script. Esta função verifica apenas o cache na memória, não o cache do arquivo.

Parâmetros

filename

O caminho para o script PHP a ser verificado.

Valor Retornado

Retorna true se filename for armazenado em cache no OPCache, false caso contrário.

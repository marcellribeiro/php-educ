filter_list

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

filter_list — Retorna a lista de todos filtros suportados

Descrição

filter_list(): array

Valor Retornado

Retorna um array dos nomes de todos filtros suportados, um array vazio se não há filtros. Índices deste array não são IDs de filtros, eles podem ser obtidos com filter_id() a partir do nome.

Exemplos

Exemplo #1 Um exemplo da filter_list()

<?php
print_r(filter_list());
?>

O exemplo acima produzirá algo semelhante a:

Array
(
    [0] => int
    [1] => boolean
    [2] => float
    [3] => validate_regexp
    [4] => validate_url
    [5] => validate_email
    [6] => validate_ip
    [7] => string
    [8] => stripped
    [9] => encoded
    [10] => special_chars
    [11] => unsafe_raw
    [12] => email
    [13] => url
    [14] => number_int
    [15] => number_float
    [16] => magic_quotes
    [17] => callback
)

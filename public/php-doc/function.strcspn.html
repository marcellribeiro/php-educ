strcspn

(PHP 4, PHP 5, PHP 7, PHP 8)

strcspn — Encontra o tamanho do segmento inicial que não contenha a máscara

Descrição

strcspn(
    string $str1,
    string $str2,
    int $start = ?,
    int $length = ?
): int

Retorna o tamanho do segmento inicial de str1 que não contém nenhum dos caracteres em str2.

Parâmetros

str1

A primeira string.

str2

A segunda string.

start

A posição inicial da string.

length

O tamanho da string.

Valor Retornado

Retorna o tamanho do segmento como um inteiro.

Notas

Nota: Esta função é binary-safe.

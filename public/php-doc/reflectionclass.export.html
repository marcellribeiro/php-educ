ReflectionClass::export

(PHP 5, PHP 7)

ReflectionClass::export — Exports a class

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 7.4.0 e foi REMOVIDA a partir do PHP 8.0.0. O uso desta função é altamente desaconselhado.

Descrição

public static ReflectionClass::export(mixed $argument, bool $return = false): string

Exports a reflected class.

Parâmetros

argument

A reflexão para exportar.

return

Definir como true retornará a exportação, em vez de emiti-la. Definir como false (o padrão) fará o oposto.

Valor Retornado

Se o parâmetro return for definido como true, então a exportação será retornada como uma string, caso contrário, null será retornado.

Exemplos

Exemplo #1 Basic usage of ReflectionClass::export()

<?php
class Apple {
public $var1;
public $var2 = 'Orange';

public function type() {
return 'Apple';
}
}
ReflectionClass::export('Apple');
?>

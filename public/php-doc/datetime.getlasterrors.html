DateTime::getLastErrors

date_get_last_errors

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

DateTime::getLastErrors -- date_get_last_errors — Retorna os avisos e erros

Descrição

Estilo orientado a objetos

public static DateTime::getLastErrors(): array

Estilo procedural

date_get_last_errors(): array

Retorna um array de avisos e erros encontrados na interpretação da string de data/hora.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna um array contendo informações sobre avisos e erros.

Exemplos

Exemplo #1 Exemplo do método DateTime::getLastErrors()

Estilo orientado a objetos

<?php
try {
$date = new DateTime('asdfasdf');
} catch (Exception $e) {
// For demonstration purposes only...
print_r(DateTime::getLastErrors());

// The real object oriented way to do this is
// echo $e->getMessage();
}
?>

Estilo procedural

<?php
$date = date_create('asdfasdf');
print_r(date_get_last_errors());
?>

Os exemplos acima produzirão:

Array
(
   [warning_count] => 1
   [warnings] => Array
       (
           [6] => Double timezone specification
       )

   [error_count] => 1
   [errors] => Array
       (
           [0] => The timezone could not be found in the database
       )

)

Os índices 6, e 0 na saída do exemplo, referem-se a posição do caractere na string onde o erro ocorreu.

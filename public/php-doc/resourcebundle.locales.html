ResourceBundle::getLocales

resourcebundle_locales

(PHP 5 >= 5.3.2, PHP 7, PHP 8, PECL intl >= 2.0.0)

ResourceBundle::getLocales -- resourcebundle_locales — Obtém as localidades suportadas

Descrição

Estilo orientado a objetos

public ResourceBundle::getLocales(string $bundlename): array

Estilo procedural

resourcebundle_locales(string $bundlename): array

Obtém as localidades disponíveis a partir do pacote informado.

Parâmetros

bundlename

Caminho do ResourceBundle de onde se obterão as localidades disponíveis, ou uma string vazia para a lista padrão de localidades.

Valor Retornado

Retorna a lista de localidades suportadas pelo pacote.

Exemplos

Exemplo #1 Exemplo da função resourcebundle_locales()

<?php
$bundle = "/user/share/data/meuapp";
echo join(PHP_EOL, resourcebundle_locales($bundle));
?>

O exemplo acima produzirá algo semelhante a:

es
root

Exemplo #2 Exemplo usando OO

<?php
$bundle = "/usr/share/data/meuapp";
$r = new ResourceBundle( 'es', $bundle);
echo join("\n", $r->getLocales($bundle));
?>

O exemplo acima produzirá algo semelhante a:

es
root

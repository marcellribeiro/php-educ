is_bool

(PHP 4, PHP 5, PHP 7, PHP 8)

is_bool — Verifica se a variável é um boleano

Descrição

is_bool(mixed $var): bool

Verifica se uma dada variável é boolean.

Parâmetros

var

The variable being evaluated.

Valor Retornado

Retorna true se var é um boolean, false caso contrário.

Exemplos

Exemplo #1 Exemplo is_bool()

<?php
$a = false;
$b = 0;

// Como $a é boolean, retornará true
if (is_bool($a) === true) {
echo 'Sim, isso é booleano';
}

// Como $b não é boolean, retornará false
if (is_bool($b) === false) {
echo 'Não, isso não é booleano';
}
?>

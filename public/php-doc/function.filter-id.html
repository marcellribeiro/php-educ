filter_id

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

filter_id — Retorna o ID de um dado nome de filtro

Descrição

filter_id(string $filtername): int

Parâmetros

filtername

Nome do filtro.

Valor Retornado

ID do filtro em caso de sucesso ou null se o filtro não existir.

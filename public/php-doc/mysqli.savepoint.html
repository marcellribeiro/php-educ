mysqli::savepoint

mysqli_savepoint

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

mysqli::savepoint -- mysqli_savepoint — Define um ponto de salvamento de transação nomeado

Descrição

Estilo orientado a objetos

public mysqli::savepoint(string $name): bool

Estilo procedural:

mysqli_savepoint(mysqli $mysql, string $name): bool

Esta função é idêntica à execução de $mysqli->query("SAVEPOINT `$name`");

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

name

O identificador do ponto de salvamento.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

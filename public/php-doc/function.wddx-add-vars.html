wddx_add_vars

(PHP 4, PHP 5, PHP 7)

wddx_add_vars — Adiciona variáveis a um pacote WDDX com o IP especificado

Descrição

wddx_add_vars(resource $packet_id, mixed $var_name, mixed $... = ?): bool

Serializa as variáveis passadas e adiciona o resultado para o pacote escolhido.

Parâmetros

Essa função recebe um número variável de parâmetros.

packet_id

Um pacote WDDX, retornado por wddx_packet_start().

var_name

Pode ser um string com o nome de uma variável ou um array contendo strings com nomes de variáveis ou até outro array, recursivamente.

...

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

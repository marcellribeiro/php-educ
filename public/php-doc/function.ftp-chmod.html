ftp_chmod

(PHP 5, PHP 7, PHP 8)

ftp_chmod — Estabelece as permissões de um arquivo via FTP

Descrição

ftp_chmod(resource $ftp_stream, int $mode, string $filename): int

Estabelece as permissões para o arquivo remoto especificado para modo.

Parâmetros

ftp_stream

O identificador da conexão FTP.

mode

As novas permisões, dadas como um valor octal.

filename

O nome do arquivo remoto.

Valor Retornado

Retorna as novas permissões em caso de sucesso, o false em caso de erro.

Exemplos

Exemplo #1 Exemplo ftp_chmod()

<?php
$file = 'public_html/index.php';

// define a conexão básica
$conn_id = ftp_connect($ftp_server);

// login com nome de usuário e senha
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// tenta usar o chmod em $file para 644
if (ftp_chmod($conn_id, 0644, $file) !== false) {
echo "Foram mudadas as permissões do arquivo $file para 644\n";
} else {
echo "não foi possível usar o chmod no arquivo $file\n";
}

// fecha a conexão
ftp_close($conn_id);
?>

odbc_procedures

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_procedures — Obtém a lista de procedures guadadas na fonte de dados especificada.

Descrição

odbc_procedures(
    resource $connection_id,
    string $qualifier = ?,
    string $owner,
    string $name
): resource

Lista todas as procedures solicitadas. Terorna um identificador de resultado contendo as informações ou false em caso de falha.

O conjunto de resultados tem as seguintes colunas:

  • PROCEDURE_QUALIFIER
  • PROCEDURE_OWNER
  • PROCEDURE_NAME
  • NUM_INPUT_PARAMS
  • NUM_OUTPUT_PARAMS
  • NUM_RESULT_SETS
  • REMARKS
  • PROCEDURE_TYPE

Os argumentos owner e name aceitam modelos de busca ('%' para encontrar zero ou mais caracteres e '_' para encontrar um único caractere).

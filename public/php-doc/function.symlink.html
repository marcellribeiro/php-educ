symlink

(PHP 4, PHP 5, PHP 7, PHP 8)

symlink — Cria um link simbólico

Descrição

symlink(string $alvo, string $link): bool

symlink() cria um link simbólico do alvo existente com o nome especificado em link.

Parâmetros

target

Alvo do link.

link

O nome do link.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Notas

Nota: Esta função não foi implementada na plataforma Windows.

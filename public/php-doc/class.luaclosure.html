A Classe LuaClosure

(PECL lua >=0.9.0)

Introdução

LuaClosure é uma classe wrapper para LUA_TFUNCTION que pode ser retornada a partir de uma chamada para uma função Lua.

Sinopse da classe

class LuaClosure {
/* Métodos */
public __invoke(mixed $arg, mixed $... = ?): void
}

Índice

  • LuaClosure::__invoke — Invoca uma closure lua

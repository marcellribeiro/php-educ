MongoDB\BSON\PackedArray::has

(mongodb >=1.16.0)

MongoDB\BSON\PackedArray::has — Returns whether a index is present in the array

Descrição

final public MongoDB\BSON\PackedArray::has(int $index): bool

Parâmetros

index (int)

The index to look for in the array.

Valor Retornado

Returns true if the index is present in the array and false otherwise.

Erros

  • Lança uma exceção MongoDB\Driver\Exception\InvalidArgumentException em caso de erro ao analisar argumentos.

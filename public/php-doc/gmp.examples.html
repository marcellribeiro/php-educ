Exemplo #1 Função factorial usando GMP

<?php
function fact($x)
{
$return = 1;
for ($i=2; $i < $x; $i++) {
$return = gmp_mul($return, $i);
}
return $return;
}

echo gmp_strval(fact(1000)) . "\n";
?>

Isto irá calcular o fatorial de 1000 (um número grande) muito rápido.

bind_textdomain_codeset

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

bind_textdomain_codeset — Define qual codificação de caractere será retornado pelas mensagens do catálogo do DOMÍNIO especificado.

Descrição

bind_textdomain_codeset(string $domain, string $codeset): string

Com bind_textdomain_codeset(), você pode definir em qual codificação será retornado a mensagem domain pela gettext() e funções similares.

Parâmetros

domain

O domínio

codeset

O conjunto de caracteres

Valor Retornado

Uma string em sucesso.

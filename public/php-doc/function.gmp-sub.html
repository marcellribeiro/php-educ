gmp_sub

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_sub — Subtrai números

Descrição

gmp_sub(resource $a, resource $b): resource

Subtrai b de a e retorna o resultado.

Parâmetros

a

The number being subtracted from.

Um objeto GMP, um int ou uma string numérica.

b

The number subtracted from a.

Um objeto GMP, um int ou uma string numérica.

Valor Retornado

Um objeto GMP.

Exemplos

Exemplo #1 Exemplo da gmp_sub()

<?php
$sub = gmp_sub("281474976710656", "4294967296"); // 2^48 - 2^32
echo gmp_strval($sub) . "\n";
?>

O exemplo acima produzirá:

281470681743360

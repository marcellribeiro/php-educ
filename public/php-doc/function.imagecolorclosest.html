imagecolorclosest

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolorclosest — Retorna o índice da cor mais próxima da cor especificada

Descrição

imagecolorclosest(
    resource $image,
    int $red,
    int $green,
    int $blue
): int

Retorna o índice da cor na paleta da imagem que seja mais "próxima" do valor RGB especificado.

A "distancia" entre a cor desejada e cada cor da paleta é calculada como se os valores RGB representassem pontos em um espaço tridimencional.

Se você criou a imagem a partir de um arquivo, apenas as cores usadas na imagem são resolvidas. Cores presentes apenas na paleta não são resolvidas.

Veja também imagecolorexact().

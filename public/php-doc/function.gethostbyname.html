gethostbyname

(PHP 4, PHP 5, PHP 7, PHP 8)

gethostbyname — Obtenha o endereço IPv4 correspondente a um determinado nome de host da Internet

Descrição

gethostbyname(string $hostname): string

Retorna o endereço IPv4 do host da Internet especificado por hostname.

Parâmetros

hostname

O nome do host.

Valor Retornado

Retorna o endereço IPv4 ou uma string contendo o endereço não modificado hostname em caso de falha.

Exemplos

Exemplo #1 Um exemplo simples gethostbyname()

<?php
$ip = gethostbyname('www.example.com');

echo $ip;
?>

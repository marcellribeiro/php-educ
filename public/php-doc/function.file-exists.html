file_exists

(PHP 4, PHP 5, PHP 7, PHP 8)

file_exists — Verifica se um arquivo ou diretório existe

Descrição

file_exists(string $filename): bool

Verifica se um arquivo ou diretório existe.

Parâmetros

filename

Caminho para o arquivo ou diretório.

No Windows, use //computername/share/filename ou \\computername\share\filename para checar arquivos em compartilhamentos de rede.

Valor Retornado

Retorna true se o arquivo ou diretório especificado por filename existe; false caso contrário.

Nota:

Esta função retorna false para links simbólicos apontando para arquivos inexistentes.

Nota:

O teste é realizado utilizando o UID/GID real, em vez de outro valor efetivo.

Nota: Como o tipo inteiro do PHP possui sinal e muitas plataformas usam números inteiros de 32 bits, algumas funções do sistema de arquivos podem retornar resultados inesperados para arquivos maiores que 2GB.

Erros

Em caso de falha, um E_WARNING será emitido.

Exemplos

Exemplo #1 Testando se um arquivo existe

<?php
$filename = '/caminho/para/arquivo.txt';

if (file_exists($filename)) {
echo "O arquivo $filename existe";
} else {
echo "O arquivo $filename não existe";
}
?>

Notas

Nota: Os resultados desta função são armazenados em cache. Consulte a função clearstatcache() para mais detalhes.

Dica

A partir do PHP 5.0.0, esta função também pode ser usada com alguns wrappers URL. Consulte Protocolos e Wrappers suportados para determinar quais wrappers suportam a família de funções stat().

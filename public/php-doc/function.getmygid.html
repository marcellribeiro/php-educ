getmygid

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

getmygid — Obtém o GID do dono do script PHP

Descrição

getmygid(): int

Obtém o ID do grupo do atual script.

Valor Retornado

Retorna o ID do grupo do atual script, ou false em erro.

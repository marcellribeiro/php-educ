md5_file

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

md5_file — Calcula hash md5 de um dado arquivo

Descrição

md5_file(string $filename, bool $raw_output = ?): string

Calcula o hash MD5 do arquivo especificado pelo parâmetro filename usando o » RSA Data Security, Inc. MD5 Message-Digest Algorithm, e retorna esse hash. A hash é um número hexadecimal de 32 caracteres.

Parâmetros

filename

O nome do arquivo

raw_output

Quando true, retorna no formato binário com o tamanho de 16. O padrão é false.

Valor Retornado

Retorna uma string em sucesso, false caso contrário.

Changelog

Versão Descrição
5.0.0 Adicionado o parâmetro raw_output
5.1.0 Modificada função para usar a API de API. Significa que você pode usá-la com wrappers, como md5_file('http://example.com/..')

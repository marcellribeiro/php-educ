imagecolorsforindex

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolorsforindex — Retorna as cores de um índice

Descrição

imagecolorsforindex(resource $image, int $index): array

Obtém a cor de um especificado índice.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

index

Valor Retornado

Retorna um array associativo com as chaves vermelho, verde, azul e alpha que contém os apropriados valores para o especificado índice de cor.

Exemplos

Exemplo #1 Exemplo imagecolorsforindex()

<?php

// abre uma imagem
$im = imagecreatefrompng('nexen.png');

// obtém uma cor
$start_x = 40;
$start_y = 50;
$color_index = imagecolorat($im, $start_x, $start_y);

// torna legível
$color_tran = imagecolorsforindex($im, $color_index);

// Qual é?
print_r($color_tran);

?>

O exemplo acima produzirá algo semelhante a:

Array
(
    [red] => 226
    [green] => 222
    [blue] => 252
    [alpha] => 0
)

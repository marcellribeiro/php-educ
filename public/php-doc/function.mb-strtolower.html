mb_strtolower

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

mb_strtolower — Torna uma string minúscula

Descrição

mb_strtolower(string $str, string $encoding = mb_internal_encoding()): string

Retorna string com todos os caracteres alfabéticos convertidos em minúsculas.

Parâmetros

string

A string a ser convertida em minúsculas.

encoding

O parâmetro encoding é a codificação de caracteres. Se for omitido ou null, o valor da codificação de caracteres interna será usado.

Valor Retornado

string com todos os caracteres alfabéticos convertidos em minúsculas.

Unicode

Para mais informações sobre as propriedades Unicode, consulte » http://www.unicode.org/reports/tr21/.

Ao contrário de strtolower(), 'alfabético' é determinado pelas propriedades de caracteres Unicode. Portanto, o comportamento desta função não é afetado por configurações de localidade e ela pode converter qualquer caractere que possui a propriedade 'alfabético', como A-trema (Ä).

Exemplos

Exemplo #1 Exemplo de mb_strtolower()

<?php
$str = "Maria Tinha Um Cordeirinho e Ela O AMAVA";
$str = mb_strtolower($str);
echo $str; // Imprime maria tinha um cordeirinho e ela o amava
?>

Exemplo #2 Exemplo de mb_strtolower() com texto UTF-8 não-Latin

<?php
$str = "Τάχιστη αλώπηξ βαφής ψημένη γη, δρασκελίζει υπέρ νωθρού κυνός";
$str = mb_strtolower($str, 'UTF-8');
echo $str; // Imprime τάχιστη αλώπηξ βαφής ψημένη γη, δρασκελίζει υπέρ νωθρού κυνός
?>

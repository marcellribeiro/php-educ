fputcsv

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

fputcsv — Formata a linha como CSV e a escreve em um ponteiro de arquivo

Descrição

fputcsv(
    resource $handle,
    array $fields,
    string $delimiter = ",",
    string $enclosure = '"',
    string $escape_char = "\"
): int

fputcsv() formata uma linha (passada como um array de campos fields) como CSV e a escreve (terminando com uma nova linha) no manipulador de arquivo especificado por handle.

Parâmetros

handle

O ponteiro de arquivo deve ser válido e deve apontar para um arquivo aberto com sucesso por fopen() ou fsockopen() (e ainda não fechado por fclose()).

fields

Um array de valores.

delimiter

O parâmetro opcional delimiter configura o caractere delimitador (apenas um caractere).

enclosure

O parâmetro opcional enclosure configura o caractere que envolve os dados (apenas um caractere).

escape_char

O parâmetro opcional escape_char configura o caractere de escape (apenas um caractere).

Valor Retornado

Retorna o comprimento da string escrita ou false em caso de falha.

Changelog

Versão Descrição
5.5.4 O parâmetro escape_char foi adicionado.

Exemplos

Exemplo #1 Exemplo da fputcsv()

<?php

$lista = array (
array('aaa', 'bbb', 'ccc', 'dddd'),
array('123', '456', '789'),
array('"aaa"', '"bbb"')
);

$fp = fopen('arquivo.csv', 'w');

foreach ($lista as $linha) {
fputcsv($fp, $linha);
}

fclose($fp);
?>

O exemplo acima irá escrever o seguinte no arquivo arquivo.csv:

aaa,bbb,ccc,dddd
123,456,789
"""aaa""","""bbb"""

Notas

Nota: Se o PHP não estiver reconhecendo apropriadamente os finais de linha ao ler arquivos em um computador Macintosh ou que foram criados em um, habilitar a opção de configuração de tempo de execução auto_detect_line_endings pode ajudar a resolver o problema.

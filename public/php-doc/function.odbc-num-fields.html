odbc_num_fields

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_num_fields — Número de colunas em um resultado

Descrição

odbc_num_fields(resource $result_id): int

Obtém o número de campos (colunas) em um resultado ODBC.

Parâmetros

result_id

O identificador de resultado retornado pela odbc_exec().

Valor Retornado

Retorna o número de campos, ou -1 em erro.

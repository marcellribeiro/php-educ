spl_autoload_call

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

spl_autoload_call — Tenta todas as funções __autoload() registradas para carregar a classe solicitada

Descrição

spl_autoload_call(string $class_name): void

Esta função pode ser usada para procurar manualmente por uma classe ou interface usando as funções __autoload registradas.

Parâmetros

class_name

O nome da classe sendo procurada.

Valor Retornado

Nenhum valor é retornado.

XSLTProcessor::getParameter

(PHP 5, PHP 7, PHP 8)

XSLTProcessor::getParameter — Obtém o valor de um parâmetro

Descrição

getParameter(string $namespaceURI, string $localName): string

Obtém o parâmetro se anteriormente foi definido pela XSLTProcessor::setParameter().

Parâmetros

namespaceURI

O namespace URI do parâmetro XSLT.

localName

O nome local do parâmetro XSLT.

Valor Retornado

O valor do parâmetro ou null se ele não é definido.

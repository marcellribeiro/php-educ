odbc_result_all

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_result_all — Mostra o resultado como uma tabela HTML

Descrição

odbc_result_all(resource $result_id, string $format = ?): int

Mostra todas as linhas de um resultado produzido pela odbc_exec(). O resultado é mostrado no formato de tabela HTML.

Parâmetros

result_id

O identificador de resultado.

format

Formatação adicional de toda tabela.

Valor Retornado

Retorna o número de linhas no resultado ou false em erro.

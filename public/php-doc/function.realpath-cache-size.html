realpath_cache_size

(PHP 5 >= 5.3.2, PHP 7, PHP 8)

realpath_cache_size — Obter o tamanho do cache do caminho absoluto

Descrição

realpath_cache_size(): int

Obtém a quantidade de memória utilizada pelo cache do caminho absoluto.

Valor Retornado

Retorna quanta memória o cache do caminho absoluto está utilizando.

Exemplos

Exemplo #1 Exemplo de realpath_cache_size()

<?php
var_dump(realpath_cache_size());
?>

O exemplo acima produzirá algo semelhante a:

int(412)

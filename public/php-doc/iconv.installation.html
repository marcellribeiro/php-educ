Instalação

Esta extensão é ativada por padrão, embora possa ser desativada compilando com --without-iconv.

A diretiva opcional --with-iconv[=DIR] é usada para especificar a localização do iconv no sistema em que o PHP está sendo compilado, caso contrário, apenas as localizações padrão são verificadas.

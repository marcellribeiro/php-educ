Introdução

A extensão DOM permite operar documentos XML através da API DOM com PHP 5.

Nota: PHP 4 oferece uma versão da extensão domxml para ser utilizada.

Nota:

A extensão DOM utiliza a codificação UTF-8. Utilize utf8_encode() e utf8_decode() para trabalhar em textos em ISO-8859-1 ou Iconv para outras codificações.

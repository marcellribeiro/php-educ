imagesavealpha

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

imagesavealpha — Define a opção para salvar a informação completa do alpha channel (ao invés de a transparencia de uma cor só) quando estiver salvando imagens PNG.

Descrição

imagesavealpha(resource $image, bool $saveflag): bool

imagesavealpha() define a opção para tentas salvar toda a informação de alpha channel (ao invés de a trnaparencia de uma cor só) quando estiver salvando imagens PNG.

Você deve desabilitar o alphablending (imagealphablending($im, false)), para poder usa-la.

Alpha channel não é suportado por todos os browsers, se você tiver problema com o seu browser, tente carregar o seu script em um browser com suporte ao alpha channel browser, ex. a versão mais recente do Mozilla.

Veja também imagealphablending().

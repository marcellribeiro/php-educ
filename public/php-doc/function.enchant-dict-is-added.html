enchant_dict_is_added

(PHP 8)

enchant_dict_is_added — Whether or not 'word' exists in this spelling-session

Descrição

enchant_dict_is_added(EnchantDictionary $dictionary, string $word): bool

Tells whether or not a word already exists in the current session.

Parâmetros

dictionary

Um dicionário Enchant retornado por enchant_broker_request_dict() ou enchant_broker_request_pwl_dict().

word

The word to lookup

Valor Retornado

Returns true if the word exists or false

Changelog

Versão Descrição
8.0.0 O parâmetro dictionary agora espera uma instância de EnchantDictionary; anteriormente, um resource era esperado.

Memcached::getOption

(PECL memcached >= 0.1.0)

Memcached::getOption — Recupera um valor de opção do Memcached

Descrição

public Memcached::getOption(int $option): mixed

Este método retorna o valor de uma option do Memcached. Algumas opções correspondem às definidas por libmemcached e algumas são específicas da extensão. Consulte Memcached Constants para obter mais informações.

Parâmetros

option

Uma das constantes Memcached::OPT_*.

Valor Retornado

Retorna o valor da opção solicitada, ou false em caso de erro.

Exemplos

Exemplo #1 Recuperando opções do Memcached

<?php
$m = new Memcached();
var_dump($m->getOption(Memcached::OPT_COMPRESSION));
var_dump($m->getOption(Memcached::OPT_POLL_TIMEOUT));
?>

O exemplo acima produzirá algo semelhante a:

bool(true)
int(1000)

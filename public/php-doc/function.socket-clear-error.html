socket_clear_error

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

socket_clear_error — Limpa o erro no socket ou o último código de erro

Descrição

socket_clear_error(resource $socket = ?): void

Essa função limpa o código de erro dado no socket ou o último erro de socket global se nenhum for especificado.

Essa função permite explicitamente resetar qualquer valor do código de erro do socket ou da extensão global do último código de erro. Isso deve ser útil para detectar dentro de uma parte da aplicação se um erro ocorreu ou não.

Parâmetros

socket

Um válido resource de socket criado com socket_create().

Valor Retornado

Nenhum valor é retornado.

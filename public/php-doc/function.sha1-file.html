sha1_file

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

sha1_file — Calcula a hash sha1 de um arquivo

Descrição

sha1_file(string $nomedoarquivo, bool $raw_output = ?): string

Calcula a hash sha1 de nomedoarquivo usando » US Secure Hash Algorithm 1, e retorna esta hash. A hash é um número hexadecimal de 40 caracteres.

Parâmetros

filename

O nome do arquivo

raw_output

Quando true, retorna no formato binário com o tamanho de 20. O padrão é false.

Valor Retornado

Retorna uma string em sucesso, false caso contrário.

ucwords

(PHP 4, PHP 5, PHP 7, PHP 8)

ucwords — Converte para maiúsculas o primeiro caractere de cada palavra

Descrição

ucwords(string $str, string $delimiters = " \t\r\n\f\v" ): string

Retorna uma string com o primeiro caractere de cada palavra em str em maiúscula, se este caractere fizer parte do alfabeto.

A definição de uma palavra é uma string de caracteres que é é precedida de qualquer caractere listado no parâmetro delimitadores (Estes são: espaço, space, form-feed, newline, carriage return, horizontal tab, e vertical tab).

Parâmetros

str

A string de entrada.

delimiters

O parâmetro opcional delimiters contém os caracteres separadores das palavras.

Valor Retornado

Retorna a string modificada.

Changelog

Versão Descrição
5.4.32, 5.5.16 O parâmetro delimiters foi adicionado.

Exemplos

Exemplo #1 Exemplo da ucwords()

<?php
$foo = 'hello world!';
$foo = ucwords($foo); // Hello World!

$bar = 'HELLO WORLD!';
$bar = ucwords($bar); // HELLO WORLD!
$bar = ucwords(strtolower($bar)); // Hello World!
?>

Exemplo #2 Exemplo de ucwords() com delimitador

<?php
$foo = 'hello|world!';
$bar = ucwords($foo); // Hello|world!

$baz = ucwords($foo, "|"); // Hello|World!
?>

Notas

Nota: Esta função é binary-safe.

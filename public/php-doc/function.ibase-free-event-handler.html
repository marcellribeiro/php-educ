ibase_free_event_handler

(PHP 5, PHP 7 < 7.4.0)

ibase_free_event_handler — Cancela um manipulador de eventos registrado

Descrição

ibase_free_event_handler(resource $event): bool

Esta função faz com que o manipulador de eventos registrado especificado por event seja cancelado. A função de callback não será mais chamada para os eventos que estava registrada para manipular.

Parâmetros

event

Um resource de evento criado com ibase_set_event_handler().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

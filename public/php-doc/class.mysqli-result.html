A classe MySQLi_Result

Introdução

Representa o conjunto de resultados da consulta feit ao banco de dados.

Sinopse da classe

class MySQLi_Result {
/* Propriedades */
int $mysqli_result->current_field;
int $mysqli_result->field_count;
?array $mysqli_result->lengths;
int|string $mysqli_result->num_rows;
/* Métodos */
mysqli_field_tell(mysqli_result $result): int
mysqli_result::data_seek(int $offset): bool
public mysqli_result::fetch_all(int $mode = MYSQLI_NUM): array
mysqli_result::fetch_assoc(): array
public mysqli_result::fetch_column(int $column = 0): null|int|float|string|false
mysqli_result::fetch_field(): object
mysqli_result::fetch_fields(): array
mysqli_result::fetch_row(): mixed
mysqli_num_fields(mysqli_result $result): int
mysqli_result::field_seek(int $fieldnr): bool
public mysqli_result::free(): void
public mysqli_result::getIterator(): Iterator
mysqli_fetch_lengths(mysqli_result $result): array|false
mysqli_num_rows(mysqli_result $result): int|string
}

Índice

  • mysqli_result::__construct — Constructs a mysqli_result object
  • mysqli_result::$current_field — Get current field offset of a result pointer
  • mysqli_result::data_seek — Ajusta o ponteiro do resultado para uma linha arbritaria no conjunto de resutados
  • mysqli_result::fetch_all — Fetch all result rows as an associative array, a numeric array, or both
  • mysqli_fetch_array — Obtem uma linha do resultado como uma matriz associativa, numérica, ou ambas
  • mysqli_result::fetch_assoc — Obtem uma linha do conjunto de resultados como uma matriz associativa
  • mysqli_result::fetch_column — Fetch a single column from the next row of a result set
  • mysqli_fetch_field_direct — Obtem meta dados para um único campo
  • mysqli_result::fetch_field — Retorna o próximo campo no conjunto de resultados
  • mysqli_result::fetch_fields — Retorna uma matriz de objetos representando os campos em um conjunto de resultados
  • mysqli_fetch_object — Retorna a linha atual do conjunto de resultados como um objeto
  • mysqli_result::fetch_row — Obtém uma linha do resultado como uma matriz numerada
  • mysqli_result::$field_count — Gets the number of fields in the result set
  • mysqli_field_seek — Move o ponteiro do resultado para um campo especificado
  • mysqli_result::free — Frees the memory associated with a result
  • mysqli_result::getIterator — Retrieve an external iterator
  • mysqli_result::$lengths — Returns the lengths of the columns of the current row in the result set
  • mysqli_result::$num_rows — Gets the number of rows in the result set

Dependências

Para as funções de E-mail estarem disponíveis, o PHP deve ter acesso ao binário sendmail de seu sistema durante a compilação. Se você utiliza um outro programa de email, tal como qmail ou postfix, certifique-se de usar os "wrappers" (envelopes) de sendmail apropriados que vêm com eles. O PHP vai procurar primeiro pelo sendmail em seu PATH, e então nos seguintes: /usr/bin:/usr/sbin:/usr/etc:/etc:/usr/ucblib:/usr/lib. É altamente recomendado ter o sendmail disponível de seu PATH. Também, o usuário que compilou o PHP deve ter permissão para acessar o binário de sendmail.

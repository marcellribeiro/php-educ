Closure::fromCallable

(PHP 7 >= 7.1.0)

Closure::fromCallable — Converte um callable em uma closure

Descrição

public static Closure::fromCallable(callable $callback): Closure

Cria e retorna uma nova função anônima a patir do callback fornecido usando o escopo atual. Esse método verifica se o callback é chamável no escopo atual e lança um TypeError se ele não for.

Nota:

A partir do PHP 8.1.0, Sintaxe de callable de primeira classe possui a mesma semântica que este método.

Parâmetros

callback

O callable para converter.

Valor Retornado

Retorna a recém-criada Closure ou lança um TypeError se o callback não é chamável no escopo atual.

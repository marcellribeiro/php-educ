phpdbg_color

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

phpdbg_color — Define a cor de certos elementos

Descrição

phpdbg_color(int $element, string $color): void

Define a color do element fornecido.

Parâmetros

element

Uma das constantes PHPDBG_COLOR_*.

color

O nome da cor. Um de white, red, green, yellow, blue, purple, cyan ou black, opcionalmente com um traço -bold ou -underline, por exemplo, white-bold ou green-underline.

Valor Retornado

Nenhum valor é retornado.

odbc_num_rows

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_num_rows — Número de linhas em um resultado

Descrição

odbc_num_rows(resource $result_id): int

Obtém o número de linhas em um resultado ODBC. Para comandos INSERT, UPDATE e DELETE odbc_num_rows() retorna o número de linhas afetadas. Para SELECT isto pode ser o número de linhas disponíveis.

Parâmetros

result_id

O identificador de resultado retornado pela odbc_exec().

Valor Retornado

Retorna o número de linhas em um resultado ODBC. Esta função retornará -1 em erro.

Notas

Nota:

Usando odbc_num_rows() para determinar o número de linhas disponível após um SELECT, será retornado -1 com muitos drivers.

ftp_rename

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_rename — Renomeia um arquivo ou diretório no servidor FTP

Descrição

ftp_rename(resource $ftp_stream, string $oldname, string $newname): bool

ftp_rename() renomeia um arquivo ou diretório no servidor FTP.

Parâmetros

ftp_stream

O identificador da conexão FTP.

oldname

O nome do arquivo/diretório anterior.

newname

O novo nome.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Exemplo ftp_rename()

<?php
$old_file = 'somefile.txt.bak';
$new_file = 'somefile.txt';

// set up basic connection
$conn_id = ftp_connect($ftp_server);

// login with username and password
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// try to rename $old_file to $new_file
if (ftp_rename($conn_id, $old_file, $new_file)) {
echo "successfully renamed $old_file to $new_file\n";
} else {
echo "There was a problem while renaming $old_file to $new_file\n";
}

// close the connection
ftp_close($conn_id);
?>

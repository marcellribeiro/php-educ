usleep

(PHP 4, PHP 5, PHP 7, PHP 8)

usleep — Atrasa a execução em milionésimos de segundo

Descrição

usleep(int $micro_seconds): void

Atrasa a execução do programa por um dado número de milionésimos de segundos.

Parâmetros

micro_seconds

Tempo de parada em milionésimos de segundos.

Valor Retornado

Nenhum valor é retornado.

Changelog

Versão Descrição
5.0.0 Esta função agora funciona em sistemas Windows.

Exemplos

Exemplo #1 Exemplo da usleep()

<?php

// hora atual
echo date('h:i:s') . "\n";

// espera dois segundos
usleep(2000000);

// de volta!
echo date('h:i:s') . "\n";

?>

O exemplo acima produzirá:

11:13:28
11:13:30

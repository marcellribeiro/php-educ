ResourceBundle::getErrorCode

resourcebundle_get_error_code

(PHP 5 >= 5.3.2, PHP 7, PHP 8, PECL intl >= 2.0.0)

ResourceBundle::getErrorCode -- resourcebundle_get_error_code — Obtém o código do último erro gerado pelo pacote.

Descrição

Estilo orientado a objetos

public ResourceBundle::getErrorCode(): int

Estilo procedural

resourcebundle_get_error_code(ResourceBundle $r): int

Obtém o código do erro gerado pela última função realizada pelo objeto pacote.

Parâmetros

r

Um objeto ResourceBundle.

Valor Retornado

Retorna o código de erro da última chamada do objeto pacote.

Exemplos

Exemplo #1 Exemplo da função resourcebundle_get_error_code()

<?php
$r = resourcebundle_create( 'es', "/usr/share/data/meuapp");
echo $r['umastring'];
if(intl_is_failure(resourcebundle_get_error_code($r))) {
report_error("Erro no pacote");
}
?>

Exemplo #2 Exemplo usando OO

<?php
$r = new ResourceBundle( 'es', "/usr/share/data/meuapp");
echo $r['umastring'];
if(intl_is_failure(ResourceBundle::getErrorCode($r))) {
report_error("Erro no pacote");
}
?>

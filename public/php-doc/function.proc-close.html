proc_close

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

proc_close — Fecha um processo aberto por proc_open() e retorna o código de saída do processo.

Descrição

proc_close(resource $process): int

proc_close() é similar a pclose() exceto que ela só funciona em processos abertos por proc_open(). proc_close() espera pelo processo terminar, e retorna o seu código de saída. Se você tiver pipes abertos para este processo, você deve usar fclose() antes de usar esta função para previnir travamentos - o processo filho não será capaz de sair enquanto pipes estiverem abertos.

Parâmetros

process

O resource proc_open() que será fechado.

Valor Retornado

Retorna o estado do processo ao terminar que foi executado.

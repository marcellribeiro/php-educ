session_abort

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

session_abort — Descarta as alterações no array da sessão e encerra a sessão

Descrição

session_abort(): void

session_abort() encerra a sessão sem salvar dados. Consequentemente, os valores originais da sessão são mantidos.

Valor Retornado

Nenhum valor é retornado.

apache_get_version

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

apache_get_version — Retorna a versão do Apache

Descrição

apache_get_version(): string|false

Retorna a versão do Apache.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Retorna a versão do Apache se tiver sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Exemplo da função apache_get_version()

<?php
$version = apache_get_version();
echo "$version\n";
?>

O exemplo acima produzirá algo semelhante a:

Apache/1.3.29 (Unix) PHP/4.3.4

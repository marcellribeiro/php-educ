PDO::commit

(PHP 5 >= 5.1.0, PHP 7, PHP 8, PECL pdo >= 0.1.0)

PDO::commit — Envia uma transação

Descrição

public PDO::commit(): bool

Envia uma transação, retornando a conexão com o banco de dados para o modo de envio automático até que a próxima chamada para PDO::beginTransaction() inicie uma nova transação.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Erros

Lança uma PDOException se não houver transação ativa.

Nota: Uma exceção é lançada mesmo quando o atributo PDO::ATTR_ERRMODE não é PDO::ERRMODE_EXCEPTION.

Exemplos

Exemplo #1 Enviando uma transação básica

<?php
/* Begin a transaction, turning off autocommit */
$dbh->beginTransaction();

/* Insert multiple records on an all-or-nothing basis */
$sql = 'INSERT INTO fruit
(name, colour, calories)
VALUES (?, ?, ?)';

$sth = $dbh->prepare($sql);

foreach ($fruits as $fruit) {
$sth->execute(array(
$fruit->name,
$fruit->colour,
$fruit->calories,
));
}

/* Commit the changes */
$dbh->commit();

/* Database connection is now back in autocommit mode */
?>

Exemplo #2 Enviando uma transação DDL

<?php
/* Begin a transaction, turning off autocommit */
$dbh->beginTransaction();

/* Change the database schema */
$sth = $dbh->exec("DROP TABLE fruit");

/* Commit the changes */
$dbh->commit();

/* Database connection is now back in autocommit mode */
?>

Nota: Nem todos os bancos de dados permitirão que as transações operem em instruções DDL: alguns gerarão erros, enquanto outros (incluindo o MySQL) enviarão automaticamente a transação após a primeira instrução DDL ter sido encontrada.

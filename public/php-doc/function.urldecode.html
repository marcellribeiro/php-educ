urldecode

(PHP 4, PHP 5, PHP 7, PHP 8)

urldecode — Decodifica uma URL codificada

Descrição

urldecode(string $str): string

Decodifica qualquer %## codificado na string. Os símbolos 'mais' ('+') são decodificados para um caractere de espaço.

Parâmetros

str

A string a ser decodificada.

Valor Retornado

Retorna a string decodificada.

Exemplos

Exemplo #1 Exemplo da urldecode()

<?php
$query = "my=apples&are=green+and+red";

foreach (explode('&', $query) as $chunk) {
$param = explode("=", $chunk);

if ($param) {
printf("Value for parameter \"%s\" is \"%s\"<br/>\n", urldecode($param[0]), urldecode($param[1]));
}
}
?>

Notas

Aviso

As superglobais $_GET e $_REQUEST já são decodificadas. Usar urldecode() em um elemento $_GET ou $_REQUEST podem gerar resultados inesperados e perigosos.

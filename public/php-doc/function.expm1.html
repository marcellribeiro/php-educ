expm1

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

expm1 — Retorna exp(numero) - 1, computado de forma que é preciso mesmo quando o valor do número é perto de zero.

Descrição

expm1(float $arg): float
Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome e documentação poderão mudar sem aviso prévio em futuras versões do PHP. Use por sua conta e risco.

expm1() retorna o equivalente a 'exp(arg) - 1' computado na forma que é exato mesmo se o valor de arg é próximo de zero, há um caso onde 'exp (arg) - 1' pode ser inexato devido a subtração de dois números que são quase iguais.

Nota: Esta função não foi implementada na plataforma Windows.

Parâmetros

arg

O argumento para processar

Valor Retornado

'e' para potência de arg menos 1

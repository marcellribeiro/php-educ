imagecolordeallocate

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolordeallocate — Desaloca uma cor para uma imagem

Descrição

imagecolordeallocate(resource $image, int $color): int

A função imagecolordeallocate() desaloca uma cor previamente alocada com imagecolorallocate() ou imagecolorallocatealpha().

<?php
$white = imagecolorallocate($im, 255, 255, 255);
imagecolordeallocate($im, $white);
?>

Veja também imagecolorallocate() e imagecolorallocatealpha().

ftp_delete

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_delete — Exclui um arquivo no servidor FTP

Descrição

ftp_delete(resource $ftp_stream, string $path): bool

ftp_delete() apaga o arquivo especificado por path no servidor FTP.

Parâmetros

ftp_stream

O identificador da conexão FTP.

path

O arquivo para excluir.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Exemplo ftp_delete()

<?php
$file = 'public_html/old.txt';

// Define a conexão básica
$conn_id = ftp_connect($ftp_server);

// login com nome de usuário e senha
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// tenta excluir $file
if (ftp_delete($conn_id, $file)) {
echo "O arquivo $file foi excluído\n";
} else {
echo "não foi possível excluir $file\n";
}

// fecha a conexão
ftp_close($conn_id);
?>

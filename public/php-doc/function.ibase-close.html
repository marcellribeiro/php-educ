ibase_close

(PHP 5, PHP 7 < 7.4.0)

ibase_close — Fecha uma conexão com um banco de dados InterBase

Descrição

ibase_close(resource $connection_id = ?): bool

Fecha a conexão com um banco de dados InterBase que esta associado ao id de conexão retornado por ibase_connect(). A transação padrão na conexão é salva, outras transações são desfeitas.

Parâmetros

connection_id

Um identificador de link InterBase retornado pela ibase_connect(). Se omitido, o último link aberto é usado.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

gnupg_clearencryptkeys

(PECL gnupg >= 0.5)

gnupg_clearencryptkeys — Removes all keys which were set for encryption before

Descrição

gnupg_clearencryptkeys(resource $identifier): bool

Parâmetros

identifier

O identificador gnupg, obtido de uma chamada a gnupg_init() ou gnupg.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Procedural gnupg_clearencryptkeys() example

<?php
$res = gnupg_init();
gnupg_clearencryptkeys($res);
?>

Exemplo #2 OO gnupg_clearencryptkeys() example

<?php
$gpg = new gnupg();
$gpg->clearencryptkeys();
?>

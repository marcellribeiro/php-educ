gmp_powm

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_powm — Eleva um número a potência com módulo

Descrição

gmp_powm(resource $base, resource $exp, resource $mod): resource

Calcula (base elevado a potência exp) módulo mod. Se exp é negativo, o resultado é indefinido.

Parâmetros

base

O número da base.

Um objeto GMP, um int ou uma string numérica.

exp

A potência positiva para elevar a base.

Um objeto GMP, um int ou uma string numérica.

mod

O módulo.

Um objeto GMP, um int ou uma string numérica.

Valor Retornado

O novo (elevado) número, como um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_powm()

<?php
$pow1 = gmp_powm("2", "31", "2147483649");
echo gmp_strval($pow1) . "\n";
?>

O exemplo acima produzirá:

2147483648

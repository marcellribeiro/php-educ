Interface Countable

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Introdução

Classes que implementam Countable podem ser usadas com a função count().

Resumo da Interface

class Countable {
/* Métodos */
public count(): int
}

Índice

  • Countable::count — Count elements of an object

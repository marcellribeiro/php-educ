odbc_longreadlen

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_longreadlen — Manipulação de colunas LONG

Descrição

odbc_longreadlen(resource $result_id, int $length): bool

Habilita o manusei de colunas LONG e LONGVARBINARY.

Parâmetros

result_id

O identificador de resultado.

length

O número de bytes retornado pelo PHP é controlado pelo parâmetro length. Se ele é definido para 0, coluna de grandes dados são passadas através do cliente.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Notas

Nota:

Manipulação de colunas LONGVARBINARY é também afetado por odbc_binmode().

phpdbg_break_method

(PHP 5 >= 5.6.3, PHP 7, PHP 8)

phpdbg_break_method — Insere um ponto de interrupção na entrada de um método

Descrição

phpdbg_break_method(string $class, string $method): void

Insere um ponto de interrupção na entrada do method fornecido da class especificada.

Parâmetros

class

O nome da classe.

method

O nome do método.

Valor Retornado

Nenhum valor é retornado.

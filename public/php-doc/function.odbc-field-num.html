odbc_field_num

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_field_num — Retorna o número da coluna

Descrição

odbc_field_num(resource $result_id, string $field_name): int

Obtém o número da coluna que corresponde ao nome do campo no dado identificador de resultado.

Parâmetros

result_id

O identificador de resultado.

field_name

O nome do campo.

Valor Retornado

Retorna o número do campo como um inteiro, ou false em erro. A numeração dos campos iniciam em 1.

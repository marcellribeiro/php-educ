session_save_path

(PHP 4, PHP 5, PHP 7, PHP 8)

session_save_path — Obtém e/ou define o caminho para armazenamento da sessão atual

Descrição

session_save_path(string $path = ?): string

session_save_path() retorna o caminho do diretório atual utilizado para salvar os dados de sessão.

Parâmetros

path

Caminho dos dados de sessão. Se informado, o caminho onde os dados são salvos será alterado. session_save_path() precisa ser chamada antes de session_start() para isto.

Nota:

Em alguns sistemas operacionais, talvez você queira especificar um caminho em um sistema de arquivos (filesystem) que possa manuzear grande quantidade de pequenos arquivos eficientemente. Por exemplo, no Linux, reiserfs pode ter performance melhor do que ext2fs.

Valor Retornado

Retorna o caminho atual do diretório usado para o armazenamento de dados.

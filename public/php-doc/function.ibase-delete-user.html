ibase_delete_user

(PHP 5, PHP 7 < 7.4.0)

ibase_delete_user — Exclui um usuário de um banco de dados seguro (apenas para o IB6 ou posterior)

Descrição

ibase_delete_user(resource $service_handle, string $user_name): bool

PHP 4 uses server, dba_user_name and dba_user_password instead of service_handle parameter.

Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

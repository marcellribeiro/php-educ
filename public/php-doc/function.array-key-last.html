array_key_last

(PHP 7 >= 7.3.0, PHP 8)

array_key_last — Obtém a última chave de um array

Descrição

array_key_last(array $array): int|string|null

Obtenha a última chave do array fornecido sem afetar o ponteiro do array interno.

Parâmetros

array

Um array.

Valor Retornado

Retorna a última key do array se o array não estiver vazio; null caso contrário.

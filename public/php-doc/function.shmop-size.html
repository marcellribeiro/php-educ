shmop_size

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

shmop_size — Pega o tamanho do bloco de memória compartilhada

Descrição

shmop_size(int $shmid): int

shmop_size() é usada para pegar o tamanho, em bytes do bloco de memória compartilhada.

Parâmetros

shmid

O identificador do bloco de memória compartilhada criado por shmop_open()

Valor Retornado

Retorna um inteiro, que representa o número de bytes de bloco ocupados pela memória compartilhada.

Exemplos

Exemplo #1 Pegando o tamanho do bloco de memória compartilhada

<?php
$shm_size = shmop_size($shm_id);
?>

Isso irá trazer o tamanho de um bloco de memória compartilhada identificado por $shm_id em $shm_size.

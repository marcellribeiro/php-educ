Memcached::touch

(PECL memcached >= 2.0.0)

Memcached::touch — Define uma nova expiração para um item

Descrição

public Memcached::touch(string $key, int $expiration): bool

Memcached::touch()define um novo valor de expiração na key fornecida.

Parâmetros

key

A chave sob a qual armazenar o valor.

expiration

O tempo de expiração padrão é 0. Veja Tempos de Expiração para mais informações.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha. Use Memcached::getResultCode() se necessário.

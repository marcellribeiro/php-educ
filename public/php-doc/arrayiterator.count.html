ArrayIterator::count

(PHP 5, PHP 7, PHP 8)

ArrayIterator::count — Conta os elementos

Descrição

public ArrayIterator::count(): int

Retorna o número de elementos no array, ou o número de atributos públicos no objeto.

Aviso

Esta função não está documentada; apenas a lista de argumentos está disponível.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

O número de elementos ou atributos públicos no array ou objeto, respectivamente.

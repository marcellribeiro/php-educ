apache_get_modules

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

apache_get_modules — Retorna uma lista de módulos carregados do Apache

Descrição

apache_get_modules(): array

Retorna uma lista de módulos carregados do Apache.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Um array de módulos carregados do Apache.

Exemplos

Exemplo #1 Exemplo da função apache_get_modules()

<?php
print_r(apache_get_modules());
?>

O exemplo acima produzirá algo semelhante a:

Array
(
    [0] => core
    [1] => http_core
    [2] => mod_so
    [3] => sapi_apache2
    [4] => mod_mime
    [5] => mod_rewrite
)

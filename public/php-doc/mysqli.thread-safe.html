mysqli::thread_safe

mysqli_thread_safe

(PHP 5, PHP 7, PHP 8)

mysqli::thread_safe -- mysqli_thread_safe — Retorna se a segurança de thread é dada ou não

Descrição

Estilo orientado a objetos

public mysqli::thread_safe(): bool

Estilo procedural

mysqli_thread_safe(): bool

Informa se a biblioteca cliente é compilada como thread-safe.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

true se a biblioteca cliente for thread-safe, caso contrário, false.

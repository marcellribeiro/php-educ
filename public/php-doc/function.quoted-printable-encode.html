quoted_printable_encode

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

quoted_printable_encode — Converte uma string de 8 bits em uma string quoted-printable

Descrição

quoted_printable_encode(string $str): string

Retorna uma string quoted-printable criada de acordo com » RFC2045, seção 6.7.

Esta função é similar a imap_8bit(), exceto que essa não requer o módulo IMAP para funcionar.

Parâmetros

str

A string de entrada.

Valor Retornado

Retorna a string codificada.

ibase_wait_event

(PHP 5, PHP 7 < 7.4.0)

ibase_wait_event — Espera por um evento ser postado no banco de dados

Descrição

ibase_wait_event(string $event_name1, string $event_name2 = ?, string $... = ?): string
ibase_wait_event(
    resource $connection,
    string $event_name1,
    string $event_name2 = ?,
    string $... = ?
): string

Esta função suspende a execução do script até um dos dos eventos especificados é postado pelo banco de dados. O nome do evento que foi postado é retornado. Esta função aceita até 15 eventos de argumento.

Parâmetros

event_name1

O nome do evento.

event_name2

...

Valor Retornado

Retorna o nome do evento que foi postado.

  • ibase_free_event_handler() - Cancela um manipulador de eventos registrado

imagedestroy

(PHP 4, PHP 5, PHP 7, PHP 8)

imagedestroy — Destrói uma imagem

Descrição

imagedestroy(resource $image): bool

imagedestroy() libera qualquer memória associada com a imagem image.

Parâmetros

image

Um objeto GdImage, retornado por uma das funções de criação de imagens, como imagecreatetruecolor().

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

A classe ZMQContext

(PECL zmq >= 0.5.0)

Introdução

Sinopse da classe

class ZMQContext {
/* Métodos */
__construct(integer $io_threads = 1, boolean $is_persistent = true)
public getOpt(string $key): mixed
public getSocket(integer $type, string $persistent_id = null, callback $on_new_socket = null): ZMQSocket
public isPersistent(): boolean
public setOpt(integer $key, mixed $value): ZMQContext
}

Índice

  • ZMQContext::__construct — Constrói um novo objeto ZMQContext
  • ZMQContext::getOpt — Obtém uma opção de contexto
  • ZMQContext::getSocket — Cria um novo socket
  • ZMQContext::isPersistent — Retorna se o contexto é persistente
  • ZMQContext::setOpt — Define uma opção de socket

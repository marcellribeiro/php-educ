mysqli::select_db

mysqli_select_db

(PHP 5, PHP 7, PHP 8)

mysqli::select_db -- mysqli_select_db — Seleciona o banco de dados padrão para consultas de banco de dados

Descrição

Estilo orientado a objetos

public mysqli::select_db(string $database): bool

Estilo procedural

mysqli_select_db(mysqli $mysql, string $database): bool

Seleciona o banco de dados padrão a ser usado ao executar consultas na conexão do banco de dados.

Nota:

Esta função deve ser usada apenas para alterar o banco de dados padrão para a conexão. Você pode selecionar o banco de dados padrão com o 4º parâmetro em mysqli_connect().

Parâmetros

mysql

Somente no estilo procedural: Um objeto mysqli retornado por mysqli_connect() ou mysqli_init().

database

O nome do banco de dados.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Erros

Se o relatório de erros da extensão mysqli estiver habilitado (MYSQLI_REPORT_ERROR) e a operação solicitada falhar, um aviso será gerado. Se, além disso, o modo for definido como MYSQLI_REPORT_STRICT, uma exceção mysqli_sql_exception será lançada.

Exemplos

Exemplo #1 mysqli::select_db() exemplo

Estilo orientado a objetos

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$mysqli = new mysqli("localhost", "my_user", "my_password", "test");

/* obtem o nome do banco de dados padrão atual */
$result = $mysqli->query("SELECT DATABASE()");
$row = $result->fetch_row();
printf("O banco de dados padrão é %s.\n", $row[0]);

/* altere o banco de dados padrão para "mundo" */
$mysqli->select_db("world");

/* obtem o nome do banco de dados padrão atual */
$result = $mysqli->query("SELECT DATABASE()");
$row = $result->fetch_row();
printf("O banco de dados padrão é %s.\n", $row[0]);

Estilo procedural

<?php

mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$link = mysqli_connect("localhost", "my_user", "my_password", "test");

/* obter o nome do banco de dados padrão atual */
$result = mysqli_query($link, "SELECT DATABASE()");
$row = mysqli_fetch_row($result);
printf("O banco de dados padrão és %s.\n", $row[0]);

/* altere o banco de dados padrão para "mundo" */
mysqli_select_db($link, "world");

/* obtem o nome do banco de dados padrão atual */
$result = mysqli_query($link, "SELECT DATABASE()");
$row = mysqli_fetch_row($result);
printf("O banco de dados padrão é %s.\n", $row[0]);

Os exemplos acima produzirão:

O banco de dados padrão é test.
O banco de dados padrão é world.

Tipos literais

Tipos literais são tipos que não apenas verificam o tipo de um valor, mas também o próprio valor. O PHP tem suporte para dois tipos literais: false a partir do PHP 8.0.0 e true a partir do PHP 8.2.0.

Aviso

Antes do PHP 8.2.0, o tipo false só podia ser usado como parte de um tipo de união.

Nota: Não é possível definir tipos literais personalizados. Considere usar uma enumeração em vez disso.

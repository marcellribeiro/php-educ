Configurações em Execução

O comportamento dessas funções é afetado pelas configurações do php.ini.

Opções de Configuração de Data/Hora

Para mais detalhes e definições dos modos PHP_INI_*, veja os Onde uma configuração deve ser definida.

Aqui está uma breve explicação das diretivas de configuração.

date.default_latitude float

A latitude padrão.

date.default_longitude float

A longitude padrão.

date.sunrise_zenith float

O ápice do nascer do sol padrão.

date.sunset_zenith float

O ápice do pôr-do-sol padrão.

date.timezone string

O fuso horário padrão utilizado por todas as funções de data/hora. Em versões anteriores ao PHP 5.4.0, isso só funcionaria se a variável de ambiente TZ não estive configurada. A ordem de precedência utilizada por cada fuso horário, se não for especificado explicitamente, é descrita na página da função date_default_timezone_get(). Veja Lista de Fusos Horários Suportados a lista de todos os fusos horários suportados.

Nota: As primeiras quatro opções de configuração são utilizadas somente pelas funções date_sunrise() e date_sunset().

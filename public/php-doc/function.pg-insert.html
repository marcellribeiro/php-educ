pg_insert

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

pg_insert — Insere um array na tabela

Descrição

pg_insert(
    resource $connection,
    string $table_name,
    array $assoc_array,
    int $options = ?
): bool

pg_insert() insere os valores de assoc_array na tabela especificada por table_name. table_name deve ter no mínimo tantas colunas quanto forem os elementos em assoc_array. Os nomes dos campos assim como os valores em table_name devem ser iguais aos índices e valores de assoc_array. Retorna true em caso de sucesso ou false em caso de falha. Se o parâmetro options for especificado, pg_insert() é aplicado em assoc_array com a opção especificada.

Exemplo #1 pg_insert

<?php
$db = pg_connect ('dbname=meubanco');
// Isto é seguro uma vez que $_POST é convertido automaticamente
$res = pg_insert($db, 'post_log', $_POST);
if ($res) {
echo "Dados POST arquivados com sucesso\n";
}
else {
echo "O usuário deve ter inserido entradas inválidas\n";
}
?>

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome e documentação poderão mudar sem aviso prévio em futuras versões do PHP. Use por sua conta e risco.

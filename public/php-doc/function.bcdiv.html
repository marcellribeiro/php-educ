bcdiv

(PHP 4, PHP 5, PHP 7, PHP 8)

bcdiv — Dividir dois números de precisão arbitrária

Descrição

bcdiv(string $num1, string $num2, ?int $scale = null): string

Divide o num1 pelo num2.

Parâmetros

num1

O número dividendo, como uma string.

num2

O número divisor, como uma string.

scale

Este parâmetro opcional é usado para definir o número de dígitos após a casa decimal do resultado. Se omitido, o padrão será a escala definida globalmente com a função bcscale() ou retornará para 0 se a escala não tiver sido definida.

Valor Retornado

Retorna o resultado da divisão como uma string, ou null se num2 é 0.

Changelog

Versão Descrição
8.0.0 scale pode ser null.

Exemplos

Exemplo #1 Exemplo da bcdiv()

<?php

echo bcdiv('105', '6.55957', 3); // 16.007

?>

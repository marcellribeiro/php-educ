link

(PHP 4, PHP 5, PHP 7, PHP 8)

link — Criando um hard link

Descrição

link(string $target, string $link): bool

link() cria um hard link.

Parâmetros

target

Alvo do link.

link

O nome do link.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Erros

A função falha e emite um E_WARNING se link já existir, ou se target não existir.

Exemplos

Exemplo #1 Criando um hard link

<?php
$target = 'source.ext'; // Esse é o arquivo existente
$link = 'newfile.ext'; // Esse é o nome do link

link($target, $link);
?>

Notas

Nota: Esta função não funcionará em arquivos remotos, pois o arquivo a ser examinado deve ser acessível a partir do sistema de arquivos do servidor.

Nota: No Windows, essa função requer o PHP rodando em nível elevado de privilégios ou com o UAC desabilitado.

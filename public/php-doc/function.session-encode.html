session_encode

(PHP 4, PHP 5, PHP 7, PHP 8)

session_encode — Codifica os dados atuais da sessão como uma sessão codificada em formato string

Descrição

session_encode(): string

session_encode() retorna uma string serializada do conteúdo atual da sessão armazenados na super global $_SESSION.

Por padrão, o método de serialização usado é interno do PHP e não é o mesmo que serialize(). O método de serialização pode ser configurado usando session.serialize_handler.

Valor Retornado

Retorna, codificado, o conteúdo da sessão atual.

Notas

Aviso

Must call session_start() before using session_encode().

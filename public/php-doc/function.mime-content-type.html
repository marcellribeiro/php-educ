mime_content_type

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

mime_content_type — Detecta o tipo MIME de um arquivo

Descrição

mime_content_type(string $filename): string

Retorna o tipo MIME de um arquivo como determinado usando a informação de do arquivo magic.mime.

Parâmetros

filename

Caminho para o arquivo.

Valor Retornado

Retorna o tipo de conteúdo no formato MIME, como text/plain ou application/octet-stream.

Exemplos

Exemplo #1 Exemplo da mime_content_type()

<?php
echo mime_content_type('php.gif') . "\n";
echo mime_content_type('test.php');
?>

O exemplo acima produzirá:

image/gif
text/plain

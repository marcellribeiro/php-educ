connection_status

(PHP 4, PHP 5, PHP 7, PHP 8)

connection_status — Retorna o status da conexão

Descrição

connection_status(): int

Veja também connection_aborted() e ignore_user_abort().

Valor Retornado

Retorna o bitfield do status da conexão, que pode ser usado com as constantes CONNECTION_XXX para determinar o status da conexão.

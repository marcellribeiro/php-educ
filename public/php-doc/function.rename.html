rename

(PHP 4, PHP 5, PHP 7, PHP 8)

rename — Renomeia um arquivo ou diretório

Descrição

rename(string $from, string $to, ?resource $context = null): bool

Tenta renomear from para to, movendo entre diretórios se necessário. Se o arquivo to já existir, ele será sobrescrito. Se renomeando um diretório e to já existir, esta função emite um warning.

Parâmetros

from

O nome anterior.

Nota:

O wrapper utilizado em from precisa ser o mesmo wrapper utilizado em to.

to

O novo nome.

Nota: No Windows, se to já existir, ele precisa ter permissão de escrita. Se não a função rename() falhará e emitirá um E_WARNING.

context

Um resource de contexto de stream.

Valor Retornado

Retorna true em caso de sucesso ou false em caso de falha.

Exemplos

Exemplo #1 Exemplo da rename()

<?php
rename("/tmp/tmp_file.txt", "/home/user/login/docs/my_file.txt");
?>

Configurações em Execução

O comportamento dessas funções é afetado pelas configurações do php.ini.

Readline Opções de Configuração
Nome Padrão Modificável Changelog
cli.pager "" PHP_INI_ALL Available since PHP 5.4.0.
cli.prompt "\\b \\> " PHP_INI_ALL Available since PHP 5.4.0.

Aqui está uma breve explicação das diretivas de configuração.

cli.pager string

External tool to display output from command line.

cli.prompt string

Command line prompt.

wddx_packet_start

(PHP 4, PHP 5, PHP 7)

wddx_packet_start — Inicializa um novo pacote WDDX com uma estrutura dentro dele

Descrição

wddx_packet_start(string $comment = ?): resource

Inicializa um novo pacote WDDX para adição incremental de variáveis. A função cria automaticamente uma definição de estrutura dentro do pacote para armazenar as variáveis.

Parâmetros

comment

Uma string opcional contendo comentários.

Valor Retornado

Retorna o ID do pacote para ser usado em outras funções ou false em caso de erro.

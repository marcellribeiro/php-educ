libxml_clear_errors

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

libxml_clear_errors — Limpa o buffer de erro da libxml

Descrição

libxml_clear_errors(): void

libxml_clear_errors() limpa o buffer de erro da libxml.

Valor Retornado

Nenhum valor é retornado.
